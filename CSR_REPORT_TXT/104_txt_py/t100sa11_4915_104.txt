致伸科技今年將出版第2本企業社會責任報告書。同
時，這本報告書也依循 GRI G4 版本編寫，並申請SGS 依
據 AA1000 第一類型高度保證等級查證，彰顯我們永續
經營的決心。致伸集團長久以來努力耕耘營運業務，
並得到良好的經營成果，各事業部與福委會等單位也
不忘進行許多回饋社會與環境保護等善盡社會責任的
活動。從2015年開始，我們希望能更有系統地從組織
及制度面實踐企業社會責任，讓致伸以永續的步伐及
遠見繼續成長。因此我們首先制定了《致伸科技股份
有限公司企業社會責任實務守則》，依據本守則編制企
業社會責任辦公室，在董事會的授權之下，專職處理
相關作業。同時，我定期召集企業社會責任辦公室彙
報各項永續策略、計劃、推動與管理之工作進度，確
保國內外營運據點能夠確實執行。而致伸關切經濟、
環境及社會面向的議題已行之有年，並且組織內相關
部門也訂定制度和措施來落實企業應盡的責任；經由
企業社會責任辦公室全面地檢視公司現況，整體規劃
致伸的企業社會責任策略，相信未來致伸能夠更加貼
近全球的脈動，及時呼應各利害關係人所關注的議
題，達到永續發展的目標。有鑑於此，企業社會責任
辦公室的運作初期秉持三大原則：( 1. ) 標竿學習、借
力使力 ( 2 ) 員工認同、由內而外 ( 3 ) 落實改變、追
求更好。在此原則下，我們將在 E ( 經濟 ) 、S ( 社會 ) 、 
G ( 環境 ) 各領域逐步推行及落實，改善現行的制度及
措施，建立系統化的風險管理流程，期望在未來三年
內，致伸在永續議題中能夠發展專屬的特色及文化。
以下依各面向，詳細說明致伸的推動規劃。
落實公司治理，誠信經營
我們認為，善盡公司治理原則將是企業社會責任的首
要工作，唯有最高管理階層的實質監督與授權，才能
有效推動永續方案執行。所以在未來企業社會責任的
近期計劃中，公司治理為最優先 的推動項目；而我們
將以國內外評鑑及調查為藍圖，定期盤點精進項目、
健全董事會制度，以進入國內標竿企業之列為目標。
員工幸福超有感
今年度我們除了持續推動的「健康職場」以外，也設
定了「安心職場」和「友善職場」這2個更高層次的員
工關懷目標，提供員工愛與尊重的歸屬感。這些讓員
工有感的福利措施，我相信將有助於吸引更多人才加
入致伸大家庭，也能保持員工團隊的穩定度，進而對
本公司的長期營運策略規劃，發揮正向支持的作用。
對地球友善的致伸
身為電子週邊產品的領導廠商，我們期許自身可以引
領電子業界，從各種電子產品中排除有毒物質的使
用。我們已經在大多數的產品中汰除汞與砷，以及使
用不含鉛的原物料。此外，我們已訂出「減少能源消
耗密度及危險廢棄物 排放」的環境友善目標，顯示本
公司擁有魄力與決心，達成較佳的能源使用效率。
未來我們也將每年持續發行此報告，定期向外 
界揭
露財務績效以外的營運成果，並參考利害關係人的回
饋，依外部評鑑及標竿企業做為實踐目標，以行動落
實永續經營之企業願景。

界定報告內容和考量面邊界的流程
我們廣泛蒐集永續指標，作為我們企業社會責
任實踐的最佳指引。並透過鑑別 ( Identifica-
tion ) 、排序 ( Prioritization ) 、確證 ( Vali-
dation ) 、檢視 ( Review ) 4大步驟鑑別出重
大考量面。首先在步驟1鑑別過程中，經由企
業社會責任辦公室召開內部會議，從歷史營運
經驗、GRI G4 考量面 ( Aspects ) 、證券交易
所公司治理評鑑指標、AA1000查證標準、ISO 
26000 社會責任指引、Bloomberg ESG Sur-
vey...等國內外永續指標挑選相關議題，並整
理一份值得包含在本報告的議題清單，又稱相
關議題 ( Relevant Topics)。
接下來執行步驟2，依據GRI G4界定報告內容
三原則將相關議題 ( Relevant Topics ) 進行排
序。原則1：依據利害關係人包容性 ( Stake-
holder Inclusiveness )，藉此釐清利害關係人，
並在本報告中說明如何回應他們的合理期望與
利益。原則2：依據重大性 ( Materiality ），
再次確認本報告所挑選的議題，是否足以造成
對經濟、環境和社會的顯著衝擊。原則3：永
續性 ( Sustainability Context ) ，揭露本公
司如何對當地或全球經濟、環境及社會發展趨
勢，所完成的改善或降低破壞。同時，我們亦
遵循GRI G4指南，在步驟1所鑑別的相關議題 
( Relevant Topics ) ，從此階段開始稱為考量
面 ( Aspects ) 。同時，我們也針對利害關係
人發放問卷，蒐集他們關心的議題。
在步驟3確證的過程中，我們依據GRI G4完整
性原則 ( Completeness ) ，以評估已經鑑別
的重大考量面。完整性原則包括範圍 ( Scope )，
本報告所有涵蓋的考量面，例如經濟績效、原
物料、職業健康與安全以及員工訓練教育；考
量面邊界 ( Aspect Boundaries ) ，對每個重大
考量面定義衝擊邊界；以及報告期間資訊的完
整性。更重要的是，本報告書經排序後的重大
考量面清單，已經過高層決策者核准。而在鑑
別重大考量面與邊界的最後階段，也就是步驟
4檢視，將在本報告發布後，展開利害關係人
的回饋資訊蒐集，為下一個報告週期做準備。
因為，各方利害關係人的意見調查結果，將有
助於下一報告週期的鑑別步驟。
重大考量面揭露邊界
在界定報告內容過程中，我們鑑別出24個重
大考量面，包括經濟類1個，環境面8個，社會
面15個。經濟類別之重大考量面為經濟績效。
環境類別重大考量面為原物料、能源、氣體排
放、廢污水與廢棄物、產品和服務、法規遵
循、整體支出與投資情況、供應商環境評估。
社會類別重大考量面為勞僱關係、勞資關係、
職業健康與安全、訓練與教育、員工多元化與
平等機會、女男同酬、勞工實務申訴機制、童
工、強迫與強制勞動、當地社區、反貪腐、當
地社會法規遵循、顧客的健康與安全、顧客隱
私、產品服務法規遵循。
針對每個重大考量面，我們也明確的界定組織
內外不同的衝擊邊界。組織內部在考量面上的
衝擊邊界包括台灣台北營運總部外，今年度也
納入大陸東莞劉屋廠、劉屋二廠、明珠廠與西
南廠。組織外部在考量面上的邊界為供應商、
客戶、非營利組織/當地社區。

回應利害關係人關注事項
我們透過問卷回收與加權統計，取得利害關係人議合所提出之關鍵議題。接著對應GRI　G4指標，
以重大考量面的方式呈現。關於我們如何回應這些關鍵議題與關注事項，以及每一關鍵議題與關注
事項的利害關係人群體，請參照本報告書鑑別重大考量面與邊界章節以及重大考量面排序統計表。 
以下是我們回應利害關係人回應的標準流程：
1. 企業社會責任辦公室小組成員分析各議題對公司的衝擊。
2. 經由議合方式，了解利害關係人對相關議題的關切程度。
3. 回收問卷，並加權統計分析，得出相關議題的重要性排序。
4. 企業社會責任辦公室向最高管理階層匯報議合結果，釐清揭露的重大性議題及內外部邊界。
5. 確認議題揭露邊界後，最終透過報告書確實回應利害關係人關注事項。
除了上述標準流程之外，我們於官網上設立『利害關係人專區』，並提供與利害關係人溝通及聯繫
的專屬信箱：CSR@primax.com.tw，對於致伸科技相關的各項議題，如有任何疑問、建議或甚至是
申訴事項，都可藉由此信箱讓我們即時回應利害關係人，保持暢通及良好的互動，而本報告期間未
有因環境、社會、人權衝擊及侵犯顧客隱私等申訴事件。

或願景藍圖，制定企業社會責任相關政策、

空氣污染檢測與防制
空氣污染會對氣候、生態系統、空氣品質、棲息地、農業及人類和動物健康產生不良影響。空氣品
質的惡化、酸化、森林退化、公共健康問題都促使地方和國際法規管制氣體的排放。因此我們定期
測量與空氣污染有關的排放數據，並致力於減少空汙氣體排放。
• 臭氧層破壞物質排放：本公司製程無產生臭氧層破壞物質排放。
• 氮氧化物、硫氧化物和其他顯著氣體的排放：本公司主要為組裝製程，屬低空氣污染負荷的製
程，所以無氮氧化物、硫氧化物的排放，其他空氣污染物皆符合當地法規要求。依當地法規規定
噴塗製程、表面貼焊技術（Surface Mount Technology，SMT）需每年執行一次環境檢測。噴塗
製程主要是苯、甲苯、二甲苯、總揮發性有機化合物（ Volatile Organic Compounds，VOC ）；注塑
製程主要是非甲烷總烴；SMT製程主要是錫及其化合物。各污染物排放量完全符合法規標準。

有害物質嚴格管理
一旦我們鑑別出產品中的有毒物質，就會立刻減用與汰除這些有毒物質，或是遴選全新供應商，開
發更多更安全的全新物料供給來源。這些努力也讓我們從製程中排除有毒物質，不僅保護工作人
員，也避免使用致伸科技產品的消費者避免健康受到危害，也保護土地、空氣與水受到污染。
展望未來，我們將更進一步針對鄰苯二甲酸酯類（Phthalate ester）擬定管控計畫。由於鄰苯二甲
酸酯類是塑膠製程中常見的塑化劑，對人體也將造成潛在危害。
目前包括歐盟及美日各國均已將此類化合物訂為優先列管物質，訂立相關法規管控，例如加州AB 
1108法案、REACH附錄17與台灣標準檢驗局CNS 4797。本公司也將列管鄰苯二甲酸酯類，持續改
善無毒原物料管理。

以人為本的員工關懷
管理方針揭露
致伸科技認為每位員工都值得應有的尊重與尊
嚴，因此我們制定並實踐平等的聘僱政策。同
時，我們也確保在我們東莞製造中心的每個人
都受到公平對待，因此密切和當地員工充分溝
通與合作，徹底根絕抵債勞工、未足齡勞工和
超時工作問題。同時，我們設定「健康職場」
的員工關懷管理方針，此外，更進一步規劃「
安心職場」和「友善職場」這2個更高層次目
標。提供員工愛與尊重的歸屬感。關於員工管
理，我們嚴格遵循當地政府的勞工法令法規，
台北營運總部遵循《勞動基準法》，東莞四廠
則遵循《勞動合同法》。並且，我們的人力資
源部門對工作場所的管理和人才發展活動，肩
負重責大任，包括人才需求多元化和跨部門人
才轉換、員工溝通、員工關懷、學習發展以及
制定薪酬和福利等激勵措施。人力資源部門定
期追蹤和分析人力資源資訊，以協助各階主管
改善工作績效，提高生產力，並以前瞻性的觀
點，提供公司高階主管審視人力資源的績效議
題。致伸科技採取開放的員工溝通政策，讓員
工能直接與所有管理層級說出他們的想法、顧
慮或相關問題。我們致力於發展一種使員工感
覺自在地詢問問題，和直接和高層主管分享他
們對企業的想法的員工溝通文化，並使用多種
開放溝通管道來執行，包括直接員工的早會、
領班座談會、每季公司內部會議、員工意見箱
等。
具競爭性的薪酬
我們的薪酬除符合當地法令外，更定期檢視市
場水準，搜集參考地薪酬當調查報告，用以規
劃及調整全公司的薪酬策略，以確保公司薪酬
競爭力高於市場，同時兼顧內部的薪酬公平
性。台北營運總部的學士與碩士學歷起薪標準
為當地基本薪資1.7~1.9倍，證明我們為了留住
優秀人才，樂於挹注資源投資員工。此外，我
們任用與敘薪是參考內部職級所對應的薪資結
構，不因性別而有不同，且我們依職級保障年
薪，共計13～14個月，更提供具競爭性的薪資
以廣納賢才。

充滿激勵的獎金制度
為激勵員工，我們訂定充滿激勵的獎酬制度，
包括年度績效獎金、固定年終獎金、5萬元以上
「金頭腦」專利獎金、久任獎金、介紹獎金、
非正式獎勵、團體季獎金、員工分紅，提供勞
苦功高的員工相對應的獎酬。
安心的退休保障
致伸的退休金福利制度皆依循各據點當地之法
律規定，台北營運總部依據台灣的勞動基準法
及勞工退休金條例，定期提撥退休準備金至法
定帳戶；東莞四廠則是依據社會保險法，為員
工投保養老保險，讓全體員工皆能於退休後享
有安心的保障。由於台灣於2005年開始實施退
休金新制，改制當時的員工可自行選擇續用舊
制或採用新制，因此於2005年6月30日(含)前入
職的員工，依法可享有舊制退休金年資，以每
位2%月薪資按月提存至勞工退休準備金專戶。
而新制退休金則依每位6%月薪資按月提撥至
員工個人退休金專戶，除僱主每月固定提撥的
6%退休金之外，員工另可依個人意願選擇提撥
0—6%的退休金至個人退休金專戶中。本次報
告期間中，致伸可支付之基金資產於退休金負
債的佔比為154%，顯示致伸足以滿足目前相關
員工退休之需求。

維護勞動人權
我們認為，所有工作同仁都應受到有尊嚴的對
待。我們也希望每位本國籍或非本國籍的工作人
員都能理解，在公平且符合道德的環境中工作，
是他們應得的權益。以此為念，我們致力於透
過以標準化流程，確實遵守基本人權正義原則實
踐。我們對於人權議題的重視，落實範疇不僅限
於台北營運總部，更擴及東莞四廠。並且針對海
內外重大投資合約，100%遵循記載人權條款的
當地勞工法規；並且我們也投入人力，確保各式
勞工規章與勞工管理皆符合人權正義原則。在此
原則下，我們遵守法律法規，禁止使用童工和強
迫勞工，不奴役或販賣勞工，尊重員工的結社
自由和集體談判權。禁止任何形式的歧視行為，
實行人性化待遇，合理安排工人的工作時間和休
息，提供合理的工資福利，持續改進。本報告期
間並未發生以下重大人權侵害事件，包括未發現
可能違反或嚴重危及結社自由及集體協商的營運
據點、未發生具有嚴重使用童工的營運據點、未
發生性騷擾事件，也沒有發生具有嚴重強迫或強
制勞動事件風險的營運據點。
平等的工作環境
我們十分重視男女平權的工作環境，因此東莞
四廠依循《勞動合同法》相關規定，台北營運總
部也配合《性別工作平等法》，100%配合當地
政府勞工政策與法令法規。除了落實性別平等，
我們的僱用、薪資及獎酬制度也遵循同工同酬原
則，不因種族、民族、社會階層、血統、宗教、
身心障礙、性別、性取向、家庭責任、婚姻狀
況、政見、年齡而有異。以下是我們的兩性平等
的相關措施。

持續的員工高效 
溝通管道
管理方針
在致伸，「以人為本」代表無論在產品的研發
和製造上，或是在照顧員工的措施上，均以「
人的需求」為出發點，為了能更貼近人的需
求，持續的高效溝通是必要的途徑，因此致伸
特別注重、也時常保持下列溝通管道的暢通。
• 人資客服代表制度（Account Service）：
我們落實人資客服代表制度（Account Ser-
vice），與各事業部建立策略夥伴之關係，
提供事業單位在管理諮詢、人事顧問、決策
幕僚等支援與服務。從招募人力開始，以內
部顧問的角色針對人力缺口，與各事業部密
切討論人力資源，助於提高招募的效率與精
準度；並在人才進入公司後，適時扮演組織
內部的公關角色，建置暢通的溝通管道、加
強互動的頻率，以積極回應員工在工作與生
活上的需求，維持緊密的員工關係，並為員
工聲音的傳遞。以走動式管理即時掌握人員
發展並貼近提供的服務與建議。
• 定期溝通會議：公司訂有員工定期溝通機
制，營運長每半年與員工座談，溝通近期
公司整體營運計畫與展望、成果及企業文化
重點；事業部最高主管每季召開部門溝通會
議，確保部內同仁掌握事業部業務概況並聆
聽員工的意見。當有影響員工權利的重大營
運變化發生，致伸科技嚴格遵守各項勞動法
令的規定，對於各項勞務條件的變更，悉照
法令規定完成通知程序。台灣「勞動基準
法」載明雇主依法規定中止勞動契約者，預
告期間：一、繼續工作三個月以上一年未滿
者，於十日前預告之；二、繼續工作一年以
上三年未滿者，於二十日前預告之；三、繼
續工作三年以上者,於三十日前預告之。
• 軟性宣導措施：包括各樓層公佈欄、mail公
告、電梯播放螢幕、廠區外文化長廊等，運
用這些措施，公司傳遞內部訊息、宣導相關
須知，讓員工清楚了解公司所推行的活動與
計劃，也提供一個相較於其他途徑來說，較
為柔性訴求的溝通管道。
• 員工申訴管道：設置員工申訴及意見信箱、
申訴電話及員工訪談等，針對各項措施提供
建議，或揭發違反道德廉潔、不合理對待、
性騷擾或其他有損公司利益的行為。接獲申
訴後由人力資源處成立專案調查小組，如申
訴內容為各項措施或建議，將請權責單位出
具改善對策；如申訴內容為舉報不法行為，
則將調查結果通知當事人及主管，並且進行
相關懲處。台北營運總部2015年無接獲員
工申訴事件；東莞四廠共計受理24件，均
已進行改善。
除了上述的溝通管道之外，依據台灣及大陸法
規、文化及環境的不同，各自發展因地制宜的
溝通方式。

PRD績效與職涯發展評估制度
致伸的績效與職涯發展評估制度（Performance reviewand development，簡稱PRD），提供員工
與主管雙向溝通機會。在此過程了解個人的工作表現以及所需要提升的工作能力，同時，配合個人
專業技能、職涯興趣以及工作風格，制定屬於個人的學習發展計劃。讓每一位致伸人都能超越自
我，開啟無限潛能。具體措施為，高階主管確立營運目標後，每個主管與所屬員工會於年初、年中
和年末與員工雙向溝通，涵蓋項目包括：
• 年度目標設定：依部門平衡計分卡（The 
Balanced ScoreCard，簡稱BSC），分別
列出主要工作目標/責任至少5項，並訂定可
量化之衡量指標，將目標依重要性給予比重
分配，做為工作表現的評核
• 核心績效才能：主管於期初勾選列入評核之
項目至少3項並給予比重分配，並於期末逐
項評核，做為職能發揮的評核
• 職能與職涯發展分析表（Career Develop-
ment/ Assessment），因應未來職涯發
展，我們於每年期末績效考核時，請員工列
出個人優勢以及需強化或提昇之才能，並透
過直屬主管的回饋，做為下一階段工作任務
與個人發展的參考方向。
值得一提的是，我們非常鼓勵同仁跨部門、跨
功能甚至跨地區輪調，以激發他們的潛能。

是專業職都擁有充分的升遷發展空間。藉由雙軌制度使每位員工充分發揮專長，同時也滿足個人的
職涯期望。讓致伸人無論在管理職或專業職，都能找到恣情揮灑的舞台。同時，我們也有內部輪調
相關機制，有效培養組織需要之多元人才，同時達到使員工在公司內部交流與拓展視野，找尋個人
興趣所在並發展能力，進而開闊職涯寬廣度，保持組織活化。
公司依人才培養需要，對表現優秀人選進行之系統性輪調
我們每年年初公告輪調方針(Guideline)，溝通當年度應輪調功能別及資格條件，人力資源部依據個
人任職資格及公司發展需要，統籌協商輪調崗位及人員，並依據各部門所提的「部門崗位輪調計
劃」媒合輪調崗位和人員，編製「集團年度輪調計劃」供指導委員會審核核定。依據核定之「年度
輪調計劃」，調任部門主管之間需進行雙向溝通並與輪調者溝通，於指定期限內提出具體的「崗
位輪調執行方案」，內容包含輪調生效時間、溝通計劃、工作移交、崗位培訓計劃、崗位導師等內
容。其後部門主管需按時執行輪調前培訓或為輪調員工安排導師、完成人事異動流程並確保員工按
照核定時間至新崗位報到。輪調執行率及成效納入部門主管年度績效目標。

管理方針
我們致力於營造自主性的學習氛圍，讓學習轉化為習慣，成為工作與生活上不可或缺的元素。而在
學習的過程中，我們非常鼓勵同仁將所知與所學相互分享交流，促進團隊與組織共同學習成長。我
們的主管們時常扮演良師益友角色，不定期的透過一對一方式，了解部屬的工作狀況，並給予最直
接、最即時的指導，隨時調整適切的學習與發展工具。 
教育訓練架構
我們的學習與發展以職能為核心架構，緊密地與公司未來發展策略與目標結合，訓練體系分為以下
三大類：
• 專業職能訓練類：專業職能方面，為強化客戶導向的專業技術能力，我們規劃研發、品管、工
程、製造、業務、採購、顧問關係管理、供應商管理等各項學習與發展項目。
• 主管才能訓練：我們依據各階層主管需具備之管理職能，規劃相對應之學習與發展項目，以強化
主管們團隊領導與策略思考能力，在組織中扮演好承上啟下的角色。
• 一般職能訓練：為了鼓勵同仁多元發展，並提高學習所帶給我們工作之外的附加價值，除了語言
類訓練，我們亦規劃通識性訓練，以激發同仁的能力與潛力，擴大學習的意義。 
多元學習管道
• 在職訓練：在工作崗位上的學習與發展，以參與工作會議、專案（任務）指派、工作輪調等方式
加強職務歷練。
• 內部訓練：分為主管訓練、專業訓練、通識性訓練三大類；包括新進人員基礎課程、各階主管課
程、專業技術課程、品質課程、通識性講座、英語課程等。
• 外部專業訓練：提供員工全額補助專業外訓費用。目的是鼓勵同仁持續自我提升專業能力，或基
於職涯發展而培養工作相關之第二專業技能。
• 海外訓練：選派工作表現優異且具發展潛力之同仁，參加短期海外專業訓練或會議，豐富國際視
野。
• 自我學習：鼓勵持續學習工作相關之知識、技能，可提出彈性工時申請，進行在職進修；此外，

更新好書推薦書單，並提供購書補助，鼓勵同仁養成讀書習慣。

針（Environment Health Safety, EHS），建立一個責任團隊和管理系統，制定嚴謹的政策、程序
及管理標準，來推動內部職安作業。我們的產品製造據點位於東莞四廠，台北營運總部則為辦公大
樓環境，沒有製造設備。根據台灣法令規定，台北營運總部設立勞工健康與安全委員會，委員會共
9人、員工代表3人，約占總人數三分之一，以協助監督和建議職業健康與安全相關規劃。今年更
以生產線上的各項安全員機制再次進行總體檢，積極透過事前預防，進而打造低工傷率及缺勤率的
工作環境，維護員工職業安全。在台北營運總部，我們實施職安教育訓練、意外事故風險管理、健
康檢查以及醫療救護。在東莞四廠，則邀請第三方驗證機構合作，針對ISO 
14001環境管理系統和
OHSAS 18001職業安全衛生管理系統執行驗證，並正式頒發查證通過證書。報告期間本公司無發生
職業病傷害相關事故，亦無重大意外死亡的情形，彰顯我們致力於打造一個環境、安全、健康工作

提供視障按摩服務
致伸大部分的員工都是需要勞心傷神的研發工程師，除了要面臨工作上的精神壓力之外，因職務需
要而長時間使用電腦，容易造成肩頸酸痛、五十肩、腕隧道症候群、眼壓過高等職業疾病的產生。
致伸僱用視障按摩師來替這些工程師鬆筋軟骨，減輕這些因長期使用電腦所帶來的不適之外，更可
以藉此紓解他們的壓力；而僱用視障按摩師除了提供員工福利之外，更提供身心障礙者一個就業的
機會，達到雙贏的目標。

擔。

到重大疾病險皆有涵蓋。
• 團體保險保障給付額高於業界水準。
• 上述保險福利措施，每年均參加專業福利調查確認目前所提供之福利水準競爭度與涵蓋度。我們
