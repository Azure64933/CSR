企業是客戶、員工、供應商或股東等，多方利害關係人交會的平台。
在 2017 年，面對升值的新台幣、人民幣等多重逆風，我們的同仁、
客戶和供應商，一起堅守崗位，助億豐再創營收高峰。對同仁付出
的感謝，我們在薪資與福利等項目表示肯定。2017 年的員工薪資和
福利，佔億豐產生的直接經濟價值之 21.1%。而支持億豐的股東，
已在 2018 年 8 月獲得新台幣 10 元 / 股的現金股利 ( 佔產生的直接
經濟價值之 14.7%)。 
我們所產銷的窗飾商品，因具光線的調節、能源效率的改善和室內
空間的美化等多項功能，廣用於居家生活或辦公室等環境。在運用
相同資源下，我們在 2017 年成功地為消費者，提供更多的優質窗
簾。窗飾產製過程中，我們除不添加有害物質，也善用無毒的可回
收材料。研發同仁的努力革新，使不織布 ( 來料 ) 生產過程，正進步
至零溶劑的膠合配方；免用 110 噸溶劑，可大幅避免不友善揮發氣
體的產生。這些持續的環境永續堅持，有助億豐在 2017 年的內 / 外
抽檢，無檢出 8 大重金屬或可溶性鉛。

透過窗簾，我們滿足客戶的需求，並改善人們的生活品質。在內部
的會議中，我們常討論品質。因我們相信，徹底做好品質，是通往
成本管控、合理交期的捷徑。所以內部的會議，行銷同仁總被要求，
從不良品、不合格品項目，開始報告。那怕是單一事件，我們的高
階經營層，也想找到原因。因那問題的根本，可能是改善或更可能
是創新的機會。我們一方面重視跨國生產，降低國際保護主義興起，
可能產生的風險；另一方面，則要求相同產品，無論中國大陸或柬
埔寨廠出產，有相同的優良品質，避免不同產地製造反衍生的客戶
困擾。
持續推進精實生產管理，是我們的達成「品質至上」的重要方略。
在精實生產的推進月會議中，我們透過廠區的現場實地了解，驗證
設備安全、平準化生產、工數抵減或浪費排除是否符合現況、理想
姿態。雖然，難免有同仁在運補作業時受撞、壓傷，或在換模 / 線
時經歷挫、割傷。但是，我們的窗簾工廠在 2017 年，並無重大傷亡、
火災事故。

我們注重管理層、人才的梯帶培育。中國大陸的當地中高階主管，
佔全部中高階主管的 85%；柬埔寨廠，則尚有不少進步的空間。中
國大陸、柬埔寨和台灣廠的 6 位工廠長中，已由 3 位青壯廠長 ( 低
於 45 歲 ) 帶領；與資深同仁混搭、友善互動，正助製程經驗、企業
文化的傳承。一起努力的結果，使億豐在 2017 年，無獲違反空汙
排放、廢水排放、廢棄物處理汙染的鉅額裁罰。
創新研發具競爭力、可區別的產品與設備，達成產品上市及量產，
是提升億豐的技術水準、競爭力的重點。為達減少爭議風險和維護
股東權益的短期、中長期目標，我們開發產品時，同步朝完整的專
利範圍佈局。我們在 2017 年，新獲 20 件專利 ( 美國、加拿大、英
國和日本專利，共 9 件 )。億豐已進一步建立內部的專利資料庫，提
供相關單位友善的專利查閱環境；我們善用外部專業律師的管道，
降低可能的侵權糾紛。
許多讀過億豐的企業社會責任報告書、年報的讀者，會發現我們的
企業社會責任報告書，用詞比較淺白。是的，因我們認為這本報告
書，是與許多不同層面關係人的重要溝通管道。動員超過 50 同仁，
我們吸收複雜、散亂的集團資料，儘量轉換成簡單的資訊或知識，
向無論非專業 / 專業、在學、在職的讀者們溝通，並回應需求。針
對專業的國際機構投資人，我們於 2017 年開始提供英文版的企業
社會責任報告書、完整財務報告，接續公佈 2017 年的英文版企業
年報，提升資訊揭露的透明度。依台灣證券交易所於 2017 年舉辦
的「第四屆公司治理評鑑」，全部受評的 864 家上市公司中，億豐
的公司治理排名在前 6-20%。
展望 2018 年，億豐面臨的經營風險，主在原材料價格的上漲壓力、
北美市場大幅改變的窗飾安全規格和全球經濟保護主義的興起等變
數。推進精實生產管理，仍將是我們的重點基調。將於 2018 年下
半年結束的固定尺寸產品無拉繩窗簾鋪貨需求，行銷、製造部門同
仁，正持續提供量販店客人所需的解決方案。我們計畫利用持續擴
增的柬埔寨產能，分散全球經濟保護主義興起帶來的不確定性。億
豐也將繼續導入新系列的窗飾商品、優化對小型窗飾零售店的行銷
與售後服務，進一步耕耘客製化市場。

億豐綜合工業 (Nien Made Enterprise, NM；股票代號 : 8464，
簡稱億豐 ) 主要製造及銷售各式的窗簾，是全球窗飾商品的領導廠
商；我們的生產基地主位於中國大陸和柬埔寨，總部位於台灣的台
中。創設於 1974 年，億豐早年於台灣中部的鹿港代工生產窗簾。
1990 年代台灣經濟的起飛，使年輕人投身傳統產業的意願降低；缺
乏勞動力成為我們發展的重要瓶頸。為突破瓶頸，我們在 1990 年
代中期於中國大陸的廣東設立工廠；中國大陸的產能擴張，適時挹
注億豐十多年的成長動能。 
 
近年，億豐的中國大陸產能已處滿載狀態；中國大陸一胎化政策造
成缺工及薪資大幅上漲問題，成為億豐中、長期發展的隱憂。面對
中國大陸的人力結構問題，億豐採跨國生產方式來降低衝擊；我們
的柬埔寨新產能已於 2014 年下半年投產。

窗簾產品依市場通路的需求，可分為透過大型連鎖店販賣的平價「固
定尺寸窗簾」，與經由小型專業窗飾零售店銷售的高價「客製化尺
寸窗簾」。於 Home Depot 或 Walmart 等大型連鎖店，消費者可
買到我們產銷的多款窗簾商品。億豐在部分北美大型連鎖店提供專
用的窗簾裁切機台，以便消費者購入當場調整窗簾幅寬 (cut- to- 
size) 的平價窗飾產品。億豐是亞洲唯一具備研發、製造、服務和維
修北美大型連鎖店專用窗簾裁切機能力的廠商，這優勢也使我們與
北美大型連鎖店的關係緊密結合。
除拓展固定尺寸的市場商機，我們自 2001 年已深耕客製化百葉門
(Shutters) 市場，並創建自有品牌 (Norman®)。於 2007 年後，億豐佈
建北美自有的 11 大區域業務中心，成功開發數千家的小型專業窗飾零
售店通路，進一步拉高競爭者的進入門檻。我們未來各項新開發的「客
製化」百葉窗 (Blinds) 及軟質窗簾 (Shades)，將可整合入現有通路進
行銷售，提高市場佔有率及獲利。

2015 年 12 月 22 日，億豐重新在台灣證券交易所上市。透
過新股發行，在外流通股數從 2.65 億股增加到 2.93 億股

業現金流入加上資本市場募集的資金，使我們的長短期借款從
2014 年的新台幣 60.3 億元，下降至 2015 年的新台幣 23.1
億。至 2016 年第 1 季，我們的長短期借款已全數清償完畢。

10 元 / 股的現金股利 (DPS，共計新台幣 29.3 億元 )。

重大主題的管理方針
專利
專利 ( 自訂重大主題 )
窗簾產品的智慧財產 ( 專利、商標、著作權或營業秘密等 ) 爭議，對億豐的利害關係人 ( 含 : 員工、客戶、
供應商與股東 ) 將造成商譽或財務損失、營運中斷、產品上市時程延後、競爭力降低等衝擊。研發方向與既
有專利技術重疊，則將造成開發的時間與人、物力的浪費。
為避免產品設計觸及法律風險，並佈局完整的專利範圍，在開發產品前，億豐的研發、智權法務部門，須
經國際專利的檢索、分析程序。針對主要競爭對手，億豐的智權法務部門定期進行專利監控作業。此外，
億豐不定期檢視既有產品之侵權風險，以降低可能的侵權糾紛。
為達減少爭議風險和維護股東權益的短期、中長期目標，億豐已進一步建立內部的專利資料庫，提供相關
單位友善的專利查閱環境，更善用外部專業律師的管道與經驗
目的與標的 : 創新研發具競爭力、可區別的產品與設備，達成產品上市及量產。降低專利糾紛的風險，並提升億豐的技術水準、競爭
力。
承諾：依各國的智慧財產權規範、億豐的內部評核規定，我們將持續專利監控作業及檢索，制定合適的研發方向，並降低專利糾紛風險。
特定的行動 : 在 2017 年，億豐針對特定競爭對手進行專利監控，共檢閱超過 2,500 筆的公開、公告專利。依「先期研究參考」、「全
新研發設計」、「上市產品檢討」、「指定個案分析」等研發創意、產品改良等不同目的，我們進行近百件的技術細部分析。
資源 : 在 2017 年，億豐的智權法務部門，對智慧財產議題，共進行 6 次內訓、5 次外訓，且對研發部門進行 2 次授課講習，加深同
仁對智慧財產保護的概念及實作技巧。 
 
管理方針有效性的評量機制 : 2017 年，億豐無新增之專利訴訟事件。我們透過企業社會責任報告書，與利害關係人溝通重大主題的
評估結果、相應調整方針。

前述的巨額罰款，按「臺灣證券交易所股份有限公司對有價證券上市公司重大訊息之查證暨公開處理程序」

目的與標的 : 符合主管機關及當地法令，是億豐的母公司及子公司永續經營的基本要求。
政策 : 按 2018 年 4 月 27 日修正的「臺灣證券交易所股份有限公司對有價證券上市公司重大訊息之查證暨公開處理程序」，正確、

承諾 : 本公司依據相關政策，定期鑑別適用之法規，並依循管理組織體系及架構，於相關作業程序中推展落實，以降低風險及避免相

顧客健康 
與安全
GRI 416
416-1 評估產品和服務類別對健康和安全的衝擊
416-2 違反有關產品與服務的健康和安全法規之事件
因不安全、有毒害的窗簾產品，對億豐的利害關係人 ( 含 : 員工、客戶、供應商與股東 ) 可造成財務、營運
中斷或商譽等衝擊風險，億豐的窗飾產品，有嚴格的製造要求及標準。億豐的中國大陸廣東、山東廠和柬
埔寨廠，生產窗飾產品過程中，皆不添加有害物質。
傳統的拉繩設計，雖消費者便於操作拉繩，調整窗簾的垂長，但可能形成線圈，對特定使用者會產生纏繞
於頸部的安全疑慮。為使產品更安全，億豐已導入多款的「無操作拉繩的窗飾設計」。億豐的「無操作拉
繩的窗飾設計」，可讓使用者輕鬆、優雅地調整窗簾的垂長；窗簾在不增加額外力量時，垂長可以維持在
使用者調整的長度並保持平衡，真正減低窗飾使用者的安全風險。2017 年，無拉繩仿木與 PVC 百葉窗佔
總百葉窗出貨量約 15%，至 2018 年首季，無拉繩百葉窗佔總出貨量已上升到 20-25%；固定尺寸產品的
無拉繩窗簾鋪貨需求，將於 2018 年下半年完畢。
目的與標的 : 億豐的經營理念，含永續經營與善盡社會責任。本公司的中國大陸廣東、山東廠和柬埔寨廠，生產窗飾產品過程中，皆
不添加有害物質。借設計與研發創新，億豐提升產品的安全及降低使用者的風險。採用無毒的可回收材料，更是公司的目標。
政策 : 億豐對生產流程有嚴格的要求及標準。例如 : 進料檢驗管理辦法 ( 最新版為 2018 年 6 月，經品管部修正 )、不合格品控制管理
程序 ( 最新版為 2018 年 6 月，經品管部修正 )、供應商管理辦法 ( 最新版為 2018 年 6 月，經生產管理部修正 ) 和產品標識管理程序 ( 最
新版為 2018 年 6 月，經品管部修正 ) 等。向供應商採購時，我們要求原材料皆需符合「法規」和「8 大重金屬含量要求」。採取內、
外部抽樣的檢測，億豐於源頭對產品的健康、安全與品質進行把關。
申訴機制 ( 營運層級 ): 在 2017 年，億豐送 SGS、TUV 等第三方檢測公司，在 REACH、RoHS 和 8 大重金屬等檢驗，未檢出應管
控的有害、有毒物質，及 8 大重金屬；也無測出可溶性鉛，檢測結果優於我們採購原材料的鉛含量標準 (Pb ≦ 90 ppm；歐洲規範
EN-71 Part 3)。於日本銷售的產品，經檢驗中心 ( 一般財団法人ボーケン品質評価機構 ) 的審核，億豐的窗簾在甲醛項目，等同 F4
顆星 (F ☆☆☆☆ ) 最高等級。對檢驗出不合格在製品、成品的生產產線，我們採取停線方式，將問題點找出並改善。除即時管理被檢
驗出的不合格製品，品管部門在成品出貨前亦進行檢驗；我們不允許將已知檢驗不合格的成品出貨。除經行銷部門管道反映，億豐的
客戶、消費者可透過客戶專區信箱 (CRM@nienmade.com.tw) 參與顧客健康與安全的申訴機制。
 
管理方針有效性的評量機制 : 2016-2017 年，億豐所產銷的產品無違反健康、法規 ( 安全、標示、市場溝通 ) 和自願性規範等違規裁罰、
警告事件。我們透過企業社會責任報告書，與利害關係人溝通重大主題的評估結果、相應調整方針。
物料
GRI 301
301-2 使用的可再生物料
窗簾產品的可再生物料使用，對億豐的利害關係人 ( 含 : 員工、客戶、供應商、股東與消費者 ) 的衝擊風險，
包含環境變遷或財務波動等。生產百葉門、百葉窗、軟質窗簾產品，主需木料、PVC、不織布等。億豐的
百葉門產品，使用經 Forest Stewardship Council (FSC) 認證的木料，並大量運用環保建材 ( 中密度纖維
板 /MDF; Medium- Density Fiberboard)，降低產品對環境變遷的衝擊。由於零組件的重量，遠輕於木材、
中密度纖維板 (MDF)、紙包裝材等可再生物料，百葉門可再生物料的使用百分比為總重量的 90+%。 
PVC(Polyvinyl chloride) 物料可由裂解石油 (Crude Oil) 衍生之副產品石油腦 (Naphta)，再裂解為乙烯
(Ethylene) 後加工產生，對環境之衝擊含不可再生資源 ( 石油 ) 的消耗。億豐的百葉窗產品，透過自廠的回
收處理 ( 如 : PVC 回收料的使用，佔總數萬噸用量的 10+%)，減少廢棄物生產量並有效運用資源。
目的與標的 : 運用相同資源下，億豐的中國大陸廣東、山東廠和柬埔寨廠為消費者提供更多的窗簾。台灣 PVC 膠管所產製之 PVC
塑膠管，透過 PVC 的回收處理，減少廢棄物生產量並有效運用資源。
責任 : 億豐由產品開發部、生產技術部、研發一部、研發二部等，負責窗簾產品可再生物料 ( 木頭、MDF、布料、其他零組件 ) 等研
發與改良。
申訴機制 ( 營運層級 ): 由於設計與生產過程工序繁複，在每年生產的數千萬窗簾產製品或數億個窗簾零組件中，我們的客戶或消費
者難免會與億豐反映產品的缺失。經由與客戶的會議，我們的行銷人員得以確認客訴的問題點，於內部每週進行檢討，回應客戶的需
求與紓解客戶的困擾。品保部收到客人品質問題的客訴後，我們工廠的生產、研發相關部門同步進行改善，以避免同樣的問題再次發
生。同時品管部的同仁會針對相關改善進行追蹤，查看問題是否己解決。輔以百葉窗、軟質窗簾等的年度客戶滿意度調查分析，我們
持續改善產品的品質及對客戶的服務。
管理方針有效性的評量機制 : 我們透過企業社會責任報告書、客戶滿意度調查分析，與利害關係人溝通重大主題的評估結果、相應調
整方針。

責任與對社會發展有所貢獻。億豐持續健全公司董事及經理人薪資
報酬制度，並落實公司治理。

利害關係人。從設廠開始，我們的柬埔寨廠已力求盡符當地法律，

市場風險
窗飾產品與房地產的景氣及交易緊密相連。我們近 80% 營收來
自北美市場，回顧 2017 年美國的新增房屋供給量，新屋開工為
120 萬戶 (+2% YoY)。此未來供給量落於歷史的中位數，對比
美國人口增長，顯示新增的房屋供給應屬相對健康。
北美房屋價格指數自全球金融海嘯 (Global Financial Crisis, 
GFC) 連年下跌，但已在 2011 年落底，房價指數從底部至 2016
年反彈達 +25%，於 2017 年，北美房價續升 +5% YoY。回昇
的房價顯示北美房屋市場的信心應正在復甦。北美房屋去化狀
況，因買、賣交易逐漸熱絡，房屋去化從高點的需要超過 10 個
月之消化時間，成交天期在 2017 年已縮短至僅 5 個月。綜觀以
上房市指標，北美房市景氣應正朝健康的回溫階段。除總經的變
化風險，我們也面臨成本上漲的不確定性。我們透過大型連鎖量
販店通路所販售的固定尺寸產品 (Ready-Made Products)，因
大型連鎖量販店易採「價格競爭策略」獲取市佔率，造成供應商
的營運壓力。例如，國際原物料成本上升，我們卻須承擔此上漲
的成本。
風險管理與機會 
除面對連鎖量販店的沉重議價壓力、訂單巨量起伏和不穩定的原
材料成本波動，我們必須透過妥善的生產調度安排，同客戶解決
問題。透過客戶的通路資訊共享、B2B 下單平台等機制，有助於
我們掌握窗飾市場變化並管理風險。

在追求經營實績成長時，我們也面對落實公司治理、發展永續環境與維護
社會公益等的企業社會責任期許。除 4 名可表達外部利害關係人意見的董

聆聽利害關係人的意見。億豐的「企業社會責任委員會」由高階主管組成，
經各營運單位 ( 如 : 管理部、 經營企劃部、財務部、稽核室和總經理室等 )
規劃、推展相關政策。利用每年出刊的「企業社會責任報告書」，我們公
開管理、推動專案的成績及公司的非財務資訊。
持續進步的公司治理
億豐的「管理部」為專職單位，推動誠信經營；「審計委員會」與「稽核
室」協助董事會，執行會計 / 財務的稽核、內部控制的實施和誠信品質的
監督。依公佈的「檢舉制度辦法」所受理之誠信、疑似舞弊議案調查案件，
我們每年至少 1 次提報「董事會」調查彙總。我們的「誠信經營守則」/
「道德行為準則」於 2015 年度訂定，並由董事會通過，明定禁止行賄及
收賄、禁止提供非法政治獻金、禁止不當慈善捐款或贊助、禁止不正當利
益、禁止侵害智慧財產權、禁止從事不公平之競爭，以強化公司治理的防
弊制度。在 2016 年，我們公佈「企業社會責任實務守則」與「公司治理
實務守則」，回應、管理利害關係人所關注的議題。
億豐持續健全公司董事及經理人薪資報酬制度，以落實公司治理。為評估
公司整體的福利、薪酬制度，億豐設「薪資報酬委員會」。於獲利時，億
豐依「公司章程」提撥不高於 1.5% 的董事酬勞、1% 至 4% 的員工酬勞；
但公司尚有累積虧損時，應先保留彌補數。「薪資報酬委員會」/「審計委
員會」成員由全體獨立董事擔任，2017 年分別召開 4 次 /6 次會議，3 位
委員皆全程參與。
為提升資訊揭露的透明度、確保海內、外機構法人 / 投資人對重大事項能
充分的知悉、參與。2015 年 12 月舉辦的「業績發表會」，透過即時的
資訊傳遞，與當日出席之 300 位利害關係人公開互動。在 2017 年，我們
共受邀參與 5 次法人說明會；400 組以上之機構法人，與我們在 2017 年
面對面地溝通。我們於 2017 年開始提供英文版的完整財務報告，接續公
佈 2017 年的英文版企業年報。於 2018 年，我們再公佈英文版的「股東
會議事規則」、「股東會開會通知」、「董事會議事規則」與「公司章
程」等，助海外的利害關係人取得完整的企業資訊，並落實公司治理。在

Magazine」採樣數千基金經理人、分析師的評比，億豐於 2017 年獲亞

底柬埔寨的人員已越過 3,000 位，我們的最低起薪
( 不含加班 ) 遠優於該國人民平均消費金額。我們相

居民的生活品質改善，有助社會往聯合國大會所設
永續發展目標 (Sustainable Development Goals, 
SDGs) 的體面工作和經濟增長 (Decent Work and 
Economic Growth) 方向前進。
