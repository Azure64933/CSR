董事長與總經理的話
創新成長 共創價值
　　回顧 2018 年，全球經濟在美中貿易關係緊張及歐洲經濟成長放緩
下，經營環境面臨逆風考驗，PC 整體市場也趨於飽和且微幅衰退，但在
電競需求持續成長，並且在相關賽事、直播平台普及下，玩家對於中高
階 PC 與周邊產品的需求提高。而自 2017 下半年起由加密貨幣帶動的繪
圖卡需求雖急速反轉，然卻使得原先陷於挖礦熱潮而供不應求的高階繪
圖卡市場回歸電競需求正軌。本公司除長期耕耘電競高階領域，持續開
發切合玩家需求的電競 PC 相關產品外，並呼應市場眾多對於內容創作需
求的企盼，推出高階影像處理系列產品。此外，在伺服器、工業電腦及
汽車電子相關 AIoT 領域皆有重大進展，也為公司成長動能注入活力。本
公司在 2018 年合併營收及獲利均創下新高，展望 2019 年，本公司將持
續展現研發、製造與銷售實力，推出全方位電競系列產品及各式商用產
品與軟體解決方案，為公司、客戶及股東創造更高的價值。
公益關懷 社會共好
　　微星以敦本務實為初衷，長期耕耘營養午餐、教育關懷、資源捐助
與弱勢協助的社會公益事業，及投入在地關懷與社區參與希冀提升對社
會影響的正能量，讓關懷的善意無限循環，充分實現「共好」的理念。
2018 年微星推動「多元教育 共好學習」計畫，在公平、合理、均等的前
提下，提供經費與設備贊助推動優質教育，除協助改善在地偏鄉教育外，
也培養學子創意發想、解決問題與動手學習的能力，提升學子自發學習
的興趣與獨立創造思考能力。報告期間該計畫協助超過 5,000 位學子與
百間學校進行教育扎根行動。
共享理念 深耕發展
　　微星近年來聚焦電競營運，除電競品牌推廣外，投入了大量資源進
行電競人才的產學合作培訓，以專業角度協助學校培訓更多電競人才，
一方面擴大電競人才來源，豐富化生態圈，二方面來深耕校園，培養學
子的品牌忠誠度，培養更多未來重度用戶。微星目前協助建置完成 5 家
學校電競教室，包括南強、僑光、啟英、萬能及十信高中，其中十信是
教育局唯一認可且指定的電子競技實驗班，除了協助學子發展興趣與擺
脫歧視外，更可提升整體產業的效率、永續性，讓產官學界都能共享經
濟價值與策略發展紅利。
積極精進 接軌國際
　　2018 年微星營運綜合績效再創佳績，在我們不斷為永續經濟發展的
同時，我們也看見接軌國際發展的必要性，包含氣候變遷、行為準則、
全球報告倡議組織準則等，以及電子製造業的特殊議題如，衝突礦產、
高風險礦產 ( 如，鎢礦 ) 與負責任商業聯盟倡議 (RBA)，這些都是微星長
期投入與關注的目標，促進以企業社會責任為本之競爭優勢，展望未來，
微星將持續本業的專注、創新與影響力，繼續在促進經濟成長的同時也
兼容社會與環境永續，為公司及利害關係人創造更高的價值。

1. 企業永續管理
1.1 微星企業社會責任承諾
　　微星在「創新智能科技」、「實踐企業公民」、「共創成長雙贏」的企業願景下，打造永續經營的企業價值，力行對社會關懷、環境保護、員工照顧、
股東承諾、客戶信任的責任與使命。在企業社會責任的落實上，由本公司董事會授權總經理暨執行長指揮各個專 ( 兼 ) 職單位，且設置永續發展辦公室
執行企業社會責任相關事宜，並定期向董事會報告，將企業社會責任融入企業願景與經營理念中。2018 年因應客戶需求與關注議題變化進行調整部分
議題關注內容，以期更具體實踐永續經營與發展企業社會責任的承諾。
企業社會責任願景
　　微星本著「健全永續治理、發展環境永續、維護社會共好」的企業
社會責任願景，期以促進經濟、社會與環境生態之平衡與永續發展，建
立良善企業公民典範。
企業社會責任使命
　　誠信治理
推行方針 : 微星重視誠信經營與良好的法治
管理，持續提升治理績效，保障相關利害
關係人權益，為社會創造正向的價值力量。
　　環境永續
推行方針 : 微星致力調整企業營運所帶來
的環境衝擊影響，追求綠色製造與供應鏈
管理，也投入改善能源耗用改善、減廢與
管制化學物質使用，作好因應氣候變遷風
險的準備。
　　社會關懷
推行方針 : 微星秉持「改變是當下開始 服
務是默默行善 貢獻是積少成多 夢想是腳踏
實地」的公益理念，在教育關懷、弱勢協
助、社區回饋與資源捐助的範疇中協助投
入金援、志工與物資，為建立共好社會持
續努力。
1.2 永續發展藍圖
　　微星在實踐永續經營的企業願景過程，透過綠色管理、永續治理、環
境永續的策略指標結合聯合國永續發展目標 (Sustainable Development 
Goals, SDGs)，持續進行發展評估且檢視與微星營運的關聯性，鑑別出 6
項對微星具有影響力的永續目標，並揭露於本報告書中。

1.4 永續發展議題鑑別
永續發展議題分析
　　微星對於企業社會責任之實踐，藉由永續治理、環境永續、維護公益與強化資訊揭露的原則下，在廣泛議題中解析出微星能力與資源所及的發展策
略，將有助於微星與利害關係人聚焦於共同的永續發展目標。
永續發展議題鑑別程序
　　•議題鑑別
　　微星除參考各項適用的永續準則 ( 如，GRI、ISO 14000 系列、RBA 等 )、投資機構 ( 如，CDP、DJSI 等 ) 與倡議議題 ( 如，SDGs、TCFD 等 ) 外，
透過業務與作業中的反饋、客戶要求與管理階層特殊議題需求，列出不同的利害關係人關注議題。為確保重大主題產出之重要性、完整性、客觀性與包
容性，亦檢視前版報告書重大性議題矩陣、標竿同業的重大主題與 GRI 準則揭露指標後，確認關注議題內容。最後篩選出 21 個永續發展議題，作為重
大主題鑑別範疇。
　　•議題排序
　　微星永續發展議題依據蒐集結果經永續發展辦公室統計後，以「責任、影響力、回應性、重要性」四項衝擊因子給與量化分析，再以議題之「對微
星之衝擊程度 ( 責任、影響力 )、利害關係人的關注程度 ( 回應性、重要性 )」進行排序，產生微星永續發展議題矩陣，取得 21 項議題。
　　•議題確認
　　為確認微星永續發展議題符合重要性、完整性、客觀性與包容性原則，由永續發展辦公室將微星永續發展議題矩陣初步鑑別結果進行檢視，並邀請
企業社會責任工作小組與主管依永續發展、GRI 準則要求、營運發展與可控作為等因素進行討論並確認微星永續發展議題矩陣。我們將上述 21 項議題
依量化結果區分為重大議題、其他受關注議題，依據鑑別結果作為報告書架構，回應利害關係人。

境及社會的永續發展努力。
風險管理組織架構
　　微星依循現行的組織架構與內控管理，建構系統性的風險管理，除
了具成本效益外，亦具備與現行管理系統結合的優點，易於落實推展管
理策略與因應措施，降低營運危機事件所發生的可能風險。
風險管控循環
　　微星在風險管控循環過程中，採取循環管理 PDCA (Plan-Do-
Check-Act) 方式進行各類風險項目評估，使公司在可控作為下採行減緩、
移轉、迴避與自承等管理措施，持續進行改善行動並與培養應變能力。

3.3 員工發展
　　人才與微星的未來發展息息相關，因此我們特別重視員工發展，並
從選才、育才、留才各方面著手，其中「育才」部份由公司內外部訓練
規劃、執行開始。針對員工進修與訓練微星科技內部訂定「教育訓練管
理辦法」。員工可經由內部訓練、外部訓練、外派等多元化的學習管道
與資源取得進修與自我發展的機會，並經由研討會、演講、公司 e 化線
上學習系統等方式學習工作專業技能與自我能力的提升，亦可利用上班
時間進行課程訓練，課程學習費用由公司全額負擔。
職涯發展願景
　　知識學習是公司能夠持續進步與發展的
關鍵，透過計畫性的職涯發展，促進知識的
變革與創新，藉以開發員工潛能、增進工作
效能、提升組織效率，促成微星的永續發展。
職涯發展藍圖
　　微星職涯是以培育優秀人才去因應電子行業的高度競爭為目的，確
立職場潛能開發、專業職能耕耘、分層培訓管理、領導才能培養為職涯
核心，藉以提升組織績效與優化組織結構，協助員工實現職涯理想，達
成微星的願景及策略。

員工是協助微星永續經營成就的基石，公司為使員工有完善的生理、心
理、情緒與職場適應，積極投入提升員工關懷、福利制度與職場安全，
帶動員工健康自主管理、培育健康職能，建構友善的支持環境，回饋與
微星共同成長的夥伴，共同追求永續未來。
樂活微星
　　微星依據同仁的興趣分別設置不同的軟硬體設備與活動規劃，平日

衡的休閒生活。
•員工協助方案
　　微星自 2010 年導入員工協助方案，提供員工免費的情緒管理、職場
壓力、法律與醫療等多元協助。員工協助方案主要能為微星帶來以下效
益 :
1. 解決組織關心的、可能影響生產力有關的議題。
2. 協助員工找出及解決影響個人工作表現的議題，如：健康、心理、家庭、
財務、酒毒癮、法律、情緒、壓力、或其他個人議題等。

已獲得妥善之處置。
•員工健檢
　　微星定期舉辦員工檢康檢查，提供優於法令的健康檢查頻率與項目，

鑑別與評估，後續再依個案狀況，透過各廠區醫護提供不同程度的健康
照護，使同仁能做好自身的健康管理。

務室，提供定期駐診醫師與每日專業護理師的健康服務。此外，我們也

人因工程管理
　　人因工程是了解人體、感知以及心智能力與限制，而應用於工具、機器、流程、方法和環境之設計，使員工能在安全舒適及合乎人性的條件下，發
揮最大效率，進而提高生產力與使用者的滿意度。為減少員工因不當的人因性危害而導致重複性肌肉骨骼傷病等職業性疾病，台北總部自 2018 年起著
手人因性危害預防調查，由環安衛管理單位配合年度健檢執行職業性肌肉骨骼傷病調查，針對調查結果進行危害評估與改善，並對於調查、評估與改善
的結果持續管控 / 追蹤，目標在針對最後的調查及分析結果，提供公司客觀性的建議與為未來廠區的人因性危害評估預作準備，以建立安全和健康的工
作場所。

為症狀可接受；對於其他 116 位有不適感的同仁已由廠區醫師立案追蹤，並已調整適當工作內容。後續將視辦理成效作為另外兩廠區的可行性評估。

供應商風險管理措施
　　在新的經濟環境下，企業廣泛採全球採購、業務外包、集中管理等
策略，這些策略都是以節省成本為核心，但無法避免增加供應鏈管理的
風險。而整體供應商鍊越來越依靠合作、共享、外包、互聯、全球化的
情況下，微星建構有彈性的供應鏈管理，依供應商的營運現況與物料配
置的分佈進行管控，減少未來遭逢經濟、環境、社會三大面向的衝擊時
供料短缺的風險。

衝突礦產管理
　　微星透過持續提升道德、環保與社會的展現，是向利害關係人呈現良善治理的基本要素。我們藉由負責任礦產倡議 (RMI) 的定期盡職調查，要求供
應商負責任採購其產品中所使用的鉭、金、錫、鎢四項金屬，避免使用衝突礦產，間接造成侵犯人權行為和違反國家或國際法的行為等問題。
　　我們基於尊重國際人權與善盡企業社會責任之立場，避免產品使用源自武裝衝突與人權侵害之礦產，除衝突礦產聲明外，內部亦完成「衝突礦產
管理辦法」的制定，以利衝突礦產之調查與管控。相關資訊說明請見永續微星網站 ( 衝突礦產政策、盡責調查措施與冶煉廠清單聲明 )：http://tw.msi.

4.5 顧客服務與滿意度
　　微星重視與品牌建立堅定關係的顧客，顧客在購買產品與購買後所
感受到的產品與服務體驗，才是品牌承諾真正獲得驗證之處。我們透過
顧客服務的鼓勵／抱怨、正面口碑/負面口碑等意見反饋，以「產品價值、
人員服務、服務環境與整體表現」來衡量顧客滿意度，創造良好的服務
體驗。
顧客服務
　　我們推動客戶參與為主軸，與顧客深度
結合的服務，透過聆聽需求、流程改善與即
時回應，培養穩健持久的顧客關係，提升顧
客忠誠度，有助於帶動客戶參與度、降低服
務成本與推升公司獲利。
•服務創新
　　微星的服務創新是建構在顧客需求上，提供更新穎、更高品質的服
務，滿足客戶的差異化需求。，凡是工作讓客戶不滿意之處，就是機會
所在，故透過下列四項服務創新思維，來超越顧客期待。
■完成顧客需求 : 包含工作需求及在執行流程上得到滿意結果，並設法找
出新的服務機會，以滿足客戶服務。
■改善客戶服務 : 透過工作步驟發掘幫助客戶完成特定核心需求，透過服
務改善，來引導新產品及服務的新領域。
■增進被服務經驗 : 協助顧客從一項服務中得到好處，不需要全新解決方
案，而是尋找各種可改進服務方式，協助顧客完成更多的工作。
■獲取最大品牌價值 : 聆聽客戶問題，這些問題可以從使用過程中找出服
務客戶的工作機會。
•產品售後服務
　　微星在全球超過 120 個國家設有服務經銷據點超過 6,500 家，提供
消費者方便的在地化產品銷售與售後服務支援，我們的服務據點可提供
換貨、檢測、技術諮詢、配件選購及筆電免費健診等優質服務 ; 全球客戶

　　人為經濟活動造成不可逆的環境影響，導致氣候變遷造成災害頻傳，及溫室氣體排放、廢水、廢棄物等環境事件衍生停工與鉅額裁罰風險，嚴重衝
擊電子製造業。為追求經營成長與突破，微星致力在產品與服務上平衡各項環保作為，提升環保與社會價值就能源、廢棄物、水資源與溫排管理等持續
進行監測與改善，朝「低碳、減廢、低污染」的產業發展行動，讓環境資產與微星發展共存，構築永續經營發展藍圖。
5.1 環境管理政策
　　本公司於營運活動中所使用的水、電、原物料等能資源、廢棄物處理、產品危害物質管制均符合廠址所在地之法令要求。為達到環境保護及永續經
營的目的，我們將持續落實與改善相關防治作業，善盡企業公民的責任，本公司承諾 :
1. 企業活動以符合環保法規及相關要求為前提。
2. 致力綠色供應鏈管理，確保環境保護有效落實。
3. 改善能資源耗用，落實節能減碳。
4. 管控廢棄物與危害性物質，落實污染持續改善。
5. 使用與發展環境友善的科技。
6. 企業良善治理，持續拓展包含環境永續的經濟發展。
環境管理目標
　　微星對各項能資源訂有環保與節能目標，透過環境管理體系進行使用、排放與減量的管理，訂下管理目標，作為執行環境政策與管理的依循，並持
續落實查核與改善。

5.2 氣候變遷管理
　　氣候變遷對電子業的衝擊，從災害的角度而言，風險分析與極端事
件衝擊為重要課題，我們在氣候與環境變遷因素之脆弱度與風險評估進
行衝擊分析，作為現有防護力的檢視，以利控管風險概況，進而尋求潛
在之發展機會。此外，微星非高耗能製造業，除了積極宣示對溫室氣體
排放予以管制，擬定並簽署承諾外，也從生活及營運過程著手，研擬降
低排放溫室氣體的管制策略，並在環境衝擊、營運衝擊上研擬因應對策。

5.5 污染防治
　　微星致力發展高品質電競產品同時，也努力提升環境品質並兼顧營
運過程所產生的污染，我們在污染防制作為上持續改進生產技術、優化
處理程序、減少環境處理成本與遵守法規要求。
空氣污染防制
　　微星的空汙源自生產製程中的輔助材料 ( 如，焊錫、助焊劑、酒精等 )
所造成的揮發性有機物 (Volatile Organic Compounds，VOCs)，我們
透過廢氣防制設備 ( 如，活性碳吸附等 ) 處理生產過程中的廢氣，所排放
氣體皆遵守廠址所在地的空氣污染防制法與空氣污染排放標準，定期監
控空氣品質 ( 含室內環境與作業環境 ) 確保空氣污染控制措施的有效性。
此外，我們在製程當中並無使用臭氧層破壞物質，亦無排放氮氧化物與
硫氧化物等氣體污染物。

電子設有中水回收系統，所回收的水資源用於灌溉植栽與廁所沖洗之用。
微星各廠址皆為處於工業區內且非生態敏感地域，廢水檢驗結果均符合
當地法令規定，對於當地周遭生態環境與相關棲息地並無嚴重之影響。

活動」，三年來在共同種植 5,200 株樹苗，參與員工超過 
400 人次，大家熱血沸騰歡喜種樹護家園。2016 年至報
告期間微星植樹成果可吸收 170,820 公斤的二氧化碳。
五股溼地生態守護計畫
五股溼地擁有北部鳥類最密集的溼地生態，透過荒野協會
努力經營耕耘，已被評鑑為國家級國家重要溼地，微星認
同荒野協會為溼地環境所做的努力，第二年提供五十萬的
贊助，作為荒野協會 2018 年度五股溼地復育計畫經費。
我們除了捐贈認養五股溼地外，對內更舉辦環境教育生活
減塑行動講座及五股溼地介紹及賞燕活動，慢慢從生活中
培養綠色習慣，期許大家珍惜我們的生活環境。
