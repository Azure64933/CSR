1 / 關於本報告
本報告資訊揭露期間為 2019 年 1 月 1 日至 2019 年 12 月 31 日，為宇瞻科技 ( Apacer ) 所出版之第 7 本企業社會責任報告書。
前一本報告的發行時間為 2019 年 8 月，每年將定期出版企業社會責任報告書，並公布於企業網站，持續對外溝通宇瞻科技 
( Apacer ) 永續作為。報告書資訊揭露範圍以宇瞻科技 ( Apacer ) 台灣營運總部與廠區為核心，不包含海外營運據點，內容 
資訊若涵蓋非台灣區相關資訊，將於報告書中備註說明。
報告範疇
本報告採用全球永續性報告協會 ( Global Reporting Initiative, GRI ) 出版之永續性報告準則 GRI Standards 版本，依循核心
選項進行編撰。透過國際公認的報告架構，完整揭露宇瞻科技 ( Apacer ) 在經濟、環境與社會領域上利害關係人所關注的議題。
報告架構
宇瞻科技 ( Apacer ) 參考國內外永續與產業趨勢，並透過重大性分析鑑別利害關係人所關注的議題，作為報告書資訊揭露
核心，同時報告書編輯遵循 GRI Standards 要求之四大報導原則與六大報告書品質原則，並以 AA1000 當責性原則標準 
( AccountAbility Principle Standards, APS ) 之重大性、包容性、回應性及衝擊性 ( 2018 年版 ) 進行撰寫。

2 / 董事長的話
董事長的話
2019 年，宇瞻開始推展品牌核心價值「說到做到．堅持 
更 好． 夥 伴 共 進 」。 鼓 勵 同 仁 以「 做 好 伴 」 的 精 神，
實 踐 聯 合 國 永 續 發 展 目 標 ( Sustainable Development 
Goals, SDGs )。並以此對應，開展宇瞻的永續策略 SSD  
( Sustainability Strategy Development )，將企業的正面 
影響力擴展到多重價值鏈。同時設定經濟、環境與社會的 
短中長期目標，定期追蹤成效與溝通改善。期望宇瞻能善盡
企業社會責任，「做」所有利害關係人的「好伴」。
回顧 2019 年，宇瞻在環境、社會與公司治理三大面向，有
以下重要作為與績效：針對環境面向，宇瞻承諾盡企業之力
守護，在 2005 年即導入 ISO 14001 環境管理系統並取得
認證，降低製程中對環境的負面衝擊以及能源使用量。此
後隨著版本更新持續優化內部管理系統，於本報告書付梓
前即完成追蹤審查認證。同時我們也導入 QC 080000:2017  
有害物質流程管理系統，預計於 2020 年配合改版提早做 
內部規劃。針對 RoHS 的要求與供應商無危害物質 ( HSF )
管理規範，宇瞻透過定期召開供應商大會進行宣導，並對 
第一階供應商進行 RBA 規範要求，對供應商做輔導改善。
還與供應商夥伴倡導重視勞工人權、環境保護與誠信透明 
經營，期能攜手合作夥伴發揮加乘效益，帶動產業鏈對於 
環境永續議題的重視。
針對社會面向，宇瞻除照顧員工之外，對在地社區發展亦
相當重視。結合 CSR 與 RBA 永續經營的精神，持續推動 
「Apacer A+ EAP ( Employee Assistance Programs ) 
員工協助方案」，在工作、家庭與健康三大領域，提供完善 
的福利制度與獎勵辦法；2019 年獲得教育部體育署頒發 
運動企業認證，透過打造優質職場留住人才並發揮品牌 
磁吸效應。有感於在地社區發展需要長期付出，宇瞻制定
「蘊綠」與「傳承」兩項永續專案。前者主要與在地 NPO
組織合作並動員企業員工，共同維護社區自然生態平衡； 
後者著重在青年培力，透過贊助田徑班與傳藝社團，一方面
協助孩子逐夢培養專長，一方面發揚與承襲傳統技藝，呼應
宇瞻 CSR 精神：「創造與分享，有溫度的記憶」，讓永續
之火不斷熠熠發光。
針對公司治理面向，宇瞻嚴守法規公開揭露資訊，以透明
化營運模式對所有利害關係人負責，2019 年首次進入公司 
治理評鑑前 5%，顯見我們在此面向的努力與現行法規一致。
而身處震盪幅度劇烈的記憶體模組產業，如何降低外部環境 
變動對公司營運造成的影響達到永續營運，是自成立以來沒有 
停止過的課題。宇瞻採用工控與消費兩種產品線雙軌策略，
當市場價格波動時可適時平衡整體營運效益，因此去年營收
相較 2018 年雖略有下滑，但毛利率提升至 18.7%，正是此
策略發揮作用的展現。而宇瞻擁有二十多年的數位儲存研發
經驗，除了既有營運範疇外，也隨著趨勢浪潮積極布局新 
市場領域，期能成為企業未來成長能量，讓中長期營運目標
除了追求獲利外，也能正面積極的回應利害關係人需求。
最後很榮幸跟大家分享，宇瞻 2018 年 CSR 報告書榮獲
TCSA 台灣企業永續報告獎 - 電子資訊與製造類組「金獎」，
這是我們連續第二年拿到如此殊榮。企業永續經營需要拓展
營運深度與廣度才能長久，在此除了感謝所有同仁的付出，
以及供應商與合作夥伴的協助，讓我們能有如此佳績；也 
勉勵同仁們能謹記我們的品牌核心價值，發揮「做好伴」
的精神，讓宇瞻不僅是值得信賴的業界領導品牌，也同時 
回應全球 CSR 浪潮，以企業自身的力量帶領合作夥伴，加入 
推動企業社會責任的行列，優化台灣企業的國際競爭力， 
成為與全球公民「做好伴」的國際永續企業。

即啟動物料備援及多站點供應鏈來減輕地緣風險，增加供應鏈彈性。

導入環境管理系統
100% 不使用衝突礦產
為邁向綠色企業，宇瞻科技 ( Apacer ) 導入品質管理 
( ISO 9001 ) 與環境管理系統 ( ISO 14001 )，依照 P-D-C-A 
( Plan-Do-Check-Act Cycle ) 管理循環，確保營運與生產
各階段符合政策規範並達成既定目標。以系統性的管理 
方法確保產品之品質與環境保證，降低產品及其製程對 
環境的負面衝擊。
自 2013 年， 宇 瞻 科 技 ( Apacer ) 根 據 無 衝 突 採 購 計 畫 
( CFSI ) 之衝突礦產調查模板 ( CMRT )，針對第一階供應商
3TG 礦源調查，所有供應商在進行基本資料建立時皆必須
填寫衝突礦產調查表。2019 年共調查 238 家供應商，其
調查結果顯示宇瞻科技 (Apacer) 相關產品所使用的金屬
礦產來自歐洲、亞洲、美洲地區，100% 無來自於非政府或
非法軍事派別在剛果民主共和國控制的地區或相鄰國家。
註：衝突礦產（Conflict Minerals）是指在武裝衝突和侵犯人權的情況下所
開採的礦物，特別是來自於剛果民主共和國及其鄰近國家所出產的錫、金、
鉭、鎢、鈷等礦產（統稱 3TG 礦產）。在這些地區所開採出的礦產是以壓榨
平民的方式，強迫在惡劣的工作環境下進行非人道採礦工作，並將採礦利潤
用於戰爭與資源掠奪上，持續的內戰使得居民的貧困程度加劇、治安狀況
惡化。

在考核方法上，宇瞻主要根據供應商的進料檢驗、生產材料異常、交期影響、外部客訴與
供應商處理態度、改善時效等項目進行評核，每次的考核結果皆會通知考核對象，並進行
溝通改善，若改善情況不佳，將會酌情減少訂購量。並依據考核分數將供應商分為 A、B、C、
D 共 4 個等級。
2018 年起，我們將 RBA 的五大規範：勞工、健康與安全、環境保護、道德規範、管理系統，
納入年度考核項目。第一階段以有工廠且具相當規模、可配合執行RBA要求之廠商優先導入。
未來將持續擴大考核對象，與供應商共同接軌國際，逐步提升供應商的永續管理能力。
目前宇瞻優先針對 A 級供應商檢討 RBA 考核結果，確認 A 級供應商皆符合 RBA 規範，未來
將會擴大 RBA 考核的檢討至 B 級供應商，若考核結果不理想，將會提供輔導與協助。

災害預防與處置
為預防產生作業災害，宇瞻科技 ( Apacer ) 制訂安全衛生管理計畫並建置職業安全系統，對
任何潛在風險進行管理。藉由執行職業安全衛生管理計畫、不定期動態稽核廠區之作業環境，
並對所有傷害、非傷害及虛驚事故展開提報追蹤改善。以消除潛在危害，期望藉由相關預防
措施，達成全年職災 1 件以下之管理目標。
宇瞻科技 ( Apacer ) 訂定詳細的「職業傷害」調查流程。調查者需會同勞工代表進行事故 
原因調查，產出之報告須經過總經理簽核，並輔導受傷同仁之健康促進。為助受傷同仁早日
返回職場，更設有漸進式復工制度，在同仁受傷無法執行職務時，由廠內醫護人員進行醫療
關懷及相關醫療建議，亦協助申請保險費用。
依據職業安全衛生法，員工具有退避權。若同仁認為其工作場所具有立即性危害，可以立刻
退出工作環境，至緊急集合點集合。平常的職場風險，皆建立相關系統進行風險評估，並 
分級執行管理與改善。若有風、水、火及地震等災害事件發生，宇瞻科技 ( Apacer ) 將成立
緊急事件應變小組，建立緊急事件應變相關辦法與程序，使危害範圍降到最低。並調查發生
原因與檢討預防措施，避免再度發生相關意外。

接觸游離輻射、有機溶劑等容易危害健康安全的工作人員，我們更實施特殊檢查項目，並
實施分級管理。
若有與工作相關之健康異常狀況，我們將進行評估與分級處置。同時經由駐廠護理師施作
「人員風險評估表」，協助個人注意健康狀況。除此之外，藉由「職場健康促進計畫」，
辨理健康促進講座與活動，更積極提供員工健康生活資訊。

移工福利政策
宇瞻科技 ( Apacer ) 一直將移工視為台灣同仁，享有台灣 
同仁所有活動及福利。自 2017 年加入 RBA 責任商業聯盟
以來，宇瞻科技 ( Apacer ) 更加深入瞭解移工來台所需的 
其他各項費用。故除延續每月提供免費宿舍、交通津貼、
伙食津貼及供應優惠 3 餐外，更協助每位移工支付海外 
一次性仲介費（含所有在職移工）、國內每月仲介服務費、
任期內各項體檢及證照費用、回程機票…等。
宇瞻科技 ( Apacer ) 建立多元的溝通管道，重視員工的
意見。目前訂有「員工申訴處理辦法」及「工作場所性騷擾
申訴及懲戒辦法」，並定期舉辦勞資協調會議，讓員工充分 
表達心聲與意見。除此之外，針對新進員工的部份，藉由
新進員工評量及訪談制度了解工作適應情況，協助盡快
融入職場環境。
所有員工。讓員工可依循公開且暢通的管道申訴勞工不公平
事件，或舉發影響公司的不法行為。
員工們需以書面資料提出申訴，受理單位為人力資源暨
行政部與稽核室，收到申訴事件後由受理單位進行初步文件
審查，並約談相關人員進行事件調查，必要時將保密申訴人
的身分。
調查結果需在 30 日內以書面證明文件回覆申訴人，申訴人
若有異議可於收到回覆後 10 日內進行再申訴。截至 2019

宇瞻科技 ( Apacer ) 明定員工申訴處理辦法，適用於公司的
會議討論結果將成為公司持續改善管理制度的重要參考 
依據。以保障全體員工的權益，促進勞資方雙向溝通順暢。
宇瞻科技 ( Apacer ) 明定「工作場所性騷擾申訴及懲戒 
辦法」，禁止任何的性騷擾行為，保障員工權益。由人力資源 
暨行政部做為性騷擾事件的申訴窗口，申訴人得要求不暴露
身分。在事件處理程序上，由申訴窗口依照事件需求召集 
「性騷擾申訴處理小組」，其中女性代表需占一半以上，並
由處理小組負責事件調查、提出裁量建議，呈請總經理決議。
若性騷擾事件調查屬實，視情節輕重將處以口頭／書面警告、 
記過、調職、減／停發獎金或除名等處分。除此之外，申訴
窗口或處理小組視需求轉介申訴者至專業機構進行心理輔導
或治療。亦配合政府政策提供性別工作平等之相關文宣及 
刊物，供員工隨時參閱。截至 2019 年底，未有性騷擾申訴
事件發生。

退休制度
宇瞻科技 ( Apacer ) 依照勞動基準法，制定員工退休辦法，明定提撥退休金、員工退休的條件及退休金給付事宜。同時依法成立「員工退休準備金監督
委員會」，以委員會名義將員工退休準備金存於法定金融機構。自 2005 年 7 月開始因應政府勞退新制，選擇新制的員工依法提撥 6% 之勞工退休金，
儲存在勞保局個人退休金專戶。選擇舊制的員工則依原辦法提撥退休準備金，保障每位員工的退休生活。

潛力人才培訓－人才供應鏈
為持續提供及培養優質人才，因應市場競爭及各項經營挑戰，宇瞻科技 ( Apacer ) 建構「人才供應鏈」機制。從
發掘儲備人才、培育為潛力人才、進而成為公司長期核心人才。人才供應鏈機制由人力資源暨行政部負責統籌
規劃及啟動，各階段人才由處級主管負責推薦、事業處主管負責初審、最後由高階主管組成的策略暨經營管理

各階段人才名單確認後，人力資源暨行政部依人才所需提供 Coach，並由 Coach 協助人才制定個人發展計畫後，
人力資源暨行政部再結合人才之職能盤點結果，為各階段人才安排所需之人才發展課程。而策略暨經營管理委員
會也會依營運策略評估人才輪調機制，本公司也依人才發展狀況回饋相對應之獎酬。

員工績效評量制度
宇瞻科技 ( Apacer ) 所有通過新訓評量的員工，包含直接人員與間接人員皆需接受績效
評量。公司的獎懲、晉升制度也依照績效評量的結果設計，讓績效評量有效連結員工
獎勵與未來發展規劃。
直接人員每月均需進行績效評量，績效評量項目包含：重大規定事項、工作品質、工作
紀律、專業知能提昇、出勤狀況五大項目。每月進行績效評量，評鑑績效等級，並發放
績效獎金，鼓勵員工提升工作效率與產能。
直接人員績效評量
間接人員每半年均需進行績效評量，績效評量項目分為「目標評量」與「職能評量」。
「目標評量」項目由單位主管與員工面談，共同於每年 1 月之前完成訂定。配合公司與
部門的目標訂定個人的年度 KPI 績效。
「職能評量」項目則參考公司訂定的職能能力對照表，評鑑員工在溝通合作、效率能力、
自我成長、領導統御等核心能力表現狀況，綜合兩者得分進行評量。年度績效評量分別於
年中與年末進行，以評估員工的工作表現，同時也藉此檢視員工的績效與個人發展計畫的
適切性。

傳承分享 - 有溫度的記憶
陣頭文化具有百年歷史，從步法、陣列到服裝都有其
含意，是早年農村時代的重要宗教儀式之一。然隨著
時間推演，陣頭文化傳承也開始出現斷層。台南藝陣
文化豐富，種類和數量都具有獨特性，而位在台南、
身處藝陣之都的大光國小傳藝社，就是希望藉由專業
藝陣老師指導培育下一代，延續並推廣陣頭文化。
宇瞻科技 ( Apacer ) 關注台灣在地傳統藝術，深知
傳承之重要性與困難，若能有企業從旁支援將更有
助於文化保留。因此從 2018 年起，持續捐款台南市
大光國小傳藝社團，透過款項贊助讓大光國小的學童
能學習傳統文化之美，讓【技藝】與【記憶】皆能被
延續與傳遞。
2019 年初，由當時的楊秀枝校長率領團員與老師，
到英國進行展演，主題為「台灣傳統：藝陣武倫敦」，
吸引僑胞與留學生約 300 人共襄盛舉，並於知名
景點倫敦橋、特拉法加廣場以及中國城等做快閃表演，
讓英國民眾與各國觀光客大為驚豔，讚嘆台灣文化的
獨特。
