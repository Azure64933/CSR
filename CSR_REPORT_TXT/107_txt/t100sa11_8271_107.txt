董事長的話
董事長的話
宇瞻科技 ( Apacer ) 2018 年邁入營運第 21 個年頭，同時重新優化品牌精神「信賴與創新」，希望用更直觀口語、
更容易理解的方式闡述。最後我們定調「做好伴」這三個字，分別代表：「說到做到．堅持更好．夥伴共進」。
希望用「做好伴」的精神，成為員工、上游廠商和下游客戶的好夥伴。
在企業永續經營層面，永續發展目標 ( SDGs )，已是全球永續議題的共同語言。2018 年，我們從 SDGs 的視角
檢視自身永續作為，盤點與規劃未來策略，擬定以企業核心競爭力為主軸，延伸宇瞻科技 ( Apacer ) 的永續策略
SSD ( Sustainability Strategy Development )，與員工、供應商等利害關係人共創嶄新的價值與成果，將企業的
正面影響力擴展到價值鏈；同時設定經濟、環境與社會的短中長期目標，並定期追蹤成效與溝通改善。
在照顧內部員工層面，我們以員工凝聚力與組織發展為基底，結合企業社會責任 ( CSR ) 與負責任商業聯盟
( Responsible Business Alliance, RBA ) 永續經營的精神，從工作 +、生活 +、健康 + 三大面向出發，提供優
質的職場環境，照顧最重要的企業經營夥伴與家人，除 2018 年「Apacer A+ EAEP ( Employee Assistance & 
Exercise Programs ) 員工協助及運動方案」獲得勞動部工作生活平衡獎的肯定外，也希望建立正向企業品牌，留
住好人才、吸引新成長動能加入，讓宇瞻持續成長。
在供應商關係層面，身為全球重要的記憶體模組製造商，我們遵守 RBA 規範，並在 2018 年首次獲得 RBA 銀級
認可。宇瞻承諾將持續推動負責任商業的價值，重視生產、環保與勞工人權。以建構永續價值鏈為首要目標，對
供應鏈夥伴倡導 CSR 永續精神，期望帶動產業共同實踐企業社會責任。
在地方社區關係層面，宇瞻科技 ( Apacer ) 積極與社區、學校、藝文團體溝通。藉由志工行動回饋鄰近社區，支
持學校體育發展，今年更擴大範疇到協助傳統技藝傳承，融合「技藝」分享「記憶」。我們明白企業營運對在地
社區的重要性，將擴大行動範圍發揮最大效益以回饋社區，尊重在地發展，創造永續共生環境。
2018 年出版之 CSR 報告書，榮獲「TCSA 台灣企業永續獎」企業永續報告類－電子資訊製造業組「金獎」殊榮，
實為宇瞻科技 ( Apacer ) 永續經營之路的新里程碑。未來我們將持續「說到做到．堅持更好．夥伴共進」的品牌
核心價值，從企業的核心競爭力出發，打造當責的企業文化，與夥伴攜手前行，以友善環境、回饋社會等方式回
應利害關係人。
總經理的話

宇瞻科技 ( Apacer ) 成立至今，一路穩定扎實的奠定良好發展根基，到 2018 年已連續 10 年穩定獲利。在瞬息萬
變的記憶體競爭市場中，我們除掌握核心技術並拓展多元應用領域外，追求永續發展的腳步也未曾停歇。2018 年
獲得「TCSA 台灣企業永續獎」的金獎認證，是肯定宇瞻科技 ( Apacer ) 持續推展 CSR 的付出，在此也分享宇瞻
科技 ( Apacer ) 近年企業社會責任的績效成果與未來策略。
以整體企業營運來看，面對動盪的 2018 年，我們繳出年營收新台幣 94.4 億、毛利率 13.87% 的成績，得到第五
屆公司治理評鑑指標前 5% 的認證，同時連續 10 年榮獲台灣精品獎佳績並獲頒成就獎，展現宇瞻科技 ( Apacer )
卓越的經營、研發和設計績效。除了深耕垂直應用與消費性數位儲存市場，因應產業趨勢也積極布局電競應用領
域與智慧物聯整合方案，除多元化發展企業營運範疇，更替未來蓄積成長能量。我們也希望中長期的營運目標在
追求獲利外，能持續以正面且實質的方式直接回應利害關係人需求。
製程與產品綠色永續是宇瞻對於環境守護的承諾。我們導入環境系統，持續降低製程中對環境的負面衝擊以及能
源使用量，並透過供應商管理落實無危害物質 ( HSF ) 規範，確保產品對於環境和人體不造成傷害。在通過負責任
商業聯盟 ( RBA ) 銀級認證後，我們對第一階供應商進行 RBA 規範，與供應商夥伴倡導重視勞工人權、環境保護
與誠信透明經營，帶動產業鏈對於永續議題的重視。
談到對內部員工的照顧，面對要求優化職場環境的社會浪潮，宇瞻除順應趨勢外，亦提供更優質的條件，堅守為
員工創造健康與友善職場環境的原則，活絡人才發展。2018 年起以「Apacer A+EAEP ( Employee Assistance & 
Exercise Programs ) 員工協助及運動方案」，從健康、家庭、工作三個面向，規劃各項福利制度，並提供多項友
善育兒協助方案，讓員工為工作衝刺之餘，也能享有健康快樂的家庭生活。
在社會回饋面向，除了連續第三年舉辦【一日志工】行動，鼓勵同仁實際投入對在地社區環境與活動的關懷。宇
瞻有感於「教育」與「文化」對永續的重要性，從 2015 年開始不間斷支持新北市育林國中田徑隊，讓青年學子
的夢想持續奔馳。2018 年起展開三年贊助台南市大光國小的計畫，將傳統「技藝」轉化為「記憶」發揚延續。我
們希望這些行動不僅是曇花一現，也期待未來能成為動力，帶動所有利害關係人共同響應與付出。
企業永續發展是一條不斷自我成長與突破的道路，如同宇瞻科技 ( Apacer ) 的品牌核心價值：「說到做到．堅持
更好．夥伴共進」，我們承諾在永續議題上持續改善，實踐「堅持做對的事」的決心，成為值得信賴的領導品牌。
在此感謝所有同仁的投入與辛勞，也希望更多台灣企業能加入推動企業社會責任的行列，優化台灣企業的國際競
爭力。

     重大性分析方法與流程
重大性分析
宇瞻科技 ( Apacer ) 為讓資訊揭露內容更符合利害關係人的期待，以企業社會責任委員會會議為內部討論的核
心，輔以問卷調查鑑別企業的重要利害關係人與重大性永續議題，作為報告書編制基礎，讓利害關係人能清楚
了解宇瞻科技 ( Apacer ) 在營運治理、環境保護、社會回饋的各項永續作為。
鑑別利害關係人 - 6 個重要利害關係人
參考產業特性、營運現況，並且採用 AA1000 利害關係人議合標準 ( Stakeholder 
Engagement Standard ) 的五大原則 ( 依賴性、責任、影響、多元觀點、張力 ) 針對利
害關係人進行排序，鑑別出 6 個宇瞻科技 ( Apacer ) 的重要利害關係人：員工、股東／
投資人、供應商、客戶、媒體、政府機關。
蒐集永續議題 - 30 個永續議題
報告書永續議題以 GRI 永續性報導準則 ( GRI 準則 ) 為基礎，參考國內外電子業之永續
作為，並經過企業社會責任委員會彙整討論後，彙整 30 個永續議題進行鑑別。
分析關注程度 - 114 份鑑別問卷
透過線上問卷的方式，了解利害關係人對於永續議題的關注程度，共發送 6 份高階管
理人問卷，以及 108 份利害關係人問卷，分別在經濟、環境、社會面向中，排序出相
關議題的影響衝擊程度及關注程度。
鑑別重點資訊 - 12 項重大性議題
以高階管理人與利害關係人的問卷調查結果為基礎，並參考產業與永續趨勢議題，以
及質化訪談結果，由企業社會責任委員會進行會議討論，最後由企業高層核准，鑑別
出 12 個重大性議題，作為 2018 年企業社會責任報告書揭露之重點資訊。

     衝突礦產
衝突礦產（Conflict Minerals）是指在武裝衝突和侵犯人權的情況下所開採的礦物，特別是來自於剛果民主共和
國及其鄰近國家所出產的錫、金、鉭、鎢 4 種礦產（統稱 3TG 礦產）。在這些地區所開採出的礦產是以壓榨平民
的方式，強迫在惡劣的工作環境下進行非人道採礦工作，並將採礦利潤用於戰爭與資源掠奪上，持續的內戰使得
居民的貧困程度加劇、治安狀況惡化。
自 2013 年，宇瞻科技 ( Apacer ) 根據無衝突採購計畫 ( CFSI ) 之衝突礦產調查模板 ( CMRT )，針對一階供應商
3TG 礦源調查，所有供應商在進行基本資料建立時皆必須填寫衝突礦產調查表。2018 年共調查 299 家供應商，
其調查結果顯示宇瞻科技 ( Apacer ) 相關產品所使用的金屬礦產來自歐洲、亞洲、美洲地區，100% 沒有來自於
非政府或非法軍事派別在剛果民主共和國控制的地區或相鄰國家。
宇瞻科技 ( Apacer ) 通過 ISO 14001 環境管理系統，建立
污染預防與改善機制、節約能源使用，降低企業營運的環
境衝擊。公司內部能資源使用以營運總部的電力與生活用
水為主，產品的製造過程不需用水，故不產生營業用廢水。
公司用水來自台灣自來水公司，水源源自石門水庫，生活
廢水由汙水管線收集至汙水處理廠處理，取水與廢水排放
並不造成顯著環境影響。

      作業環境測定
      作業安全管制
宇瞻科技 ( Apacer ) 定期委由合格的作業環境測定機構實施作業環境測定，測定項目包括粉塵、有機溶劑、二
氧化碳、照明，一旦測定結果有異常狀況，立即改善、矯正以保障人員健康。
為維護作業環境安全，宇瞻科技 ( Apacer ) 依據職業安全衛生法及安全風險評估結果、對於高架作業、動火作
業、游離輻射設備，實施特殊作業管制及工作許可。同時依法實施自動檢查，所有機械設備均依法定期檢查核
可，操作人員取得專業執照並辦理定期在職回訓。
個人安全防護方面，為防止作業災害，依照作業性質，作業人員須依規定配戴防護器具。此外，為確保作業人
員不受輻射危害，確實執行設備定期偵測工作，並規定工作人員配戴輻射佩章及參加輻射作業健康檢查，以確
實掌握作業人員接受輻射的劑量以及健康狀態。
為預防作業災害的產生，宇瞻科技 ( Apacer ) 制訂
安全衛生管理方案或控制風險措施，實施安全衛生
教育訓練計畫、不定期巡查廠區之作業環境。並且
針對所有傷害、非傷害及虛驚事故展開提報追蹤改
善，以消除潛在危害，期望藉由以上預防措施，達
成全年職災 1 件以下之管理目標。
若有災害事件發生，將成立緊急事件應變小組，建
立緊急事件應變相關辦法與程序，使危害範圍降到
最低，並調查發生原因與檢討預防措施，避免再度
發生相關意外。

      健康管理

辨理健康促進講座與活動，更積極提供員工健康生活資訊。

      2018 年 A+ EAEP - 妊娠員工協助計畫
宇瞻科技 ( Apacer ) 2018 年以「友善家庭，安心育兒」為主軸，提供員工子
女全方位的照護。為支持員工照顧子女，我們發給每位妊娠員工每月 6,000
元的育兒津貼至滿 6 個月大，如父母皆為員工或子女為雙胞胎，還加給

更提供妊娠員工專屬座椅、專業護理師評估孕婦風險辨識與控制，定期安排
醫師面談給予健康指導與衛教，減緩懷孕期間的身體負擔。妊娠員工可依身
心狀況，提出調整或縮短工作時間或更換工作內容。辦公室也有設備完善、
具有隱密性的哺集乳室，讓同仁安心使用。
另對於申請育嬰留職停薪的員工，於重返職場前會進行復職前關懷及協助，
並提供復工建議，讓新手爸媽員工在獲得充分支持與協助下順利重返職場。
並於職場內打造優質臨時兒童照護空間，碰到學校臨時停課時，員工可放心
在公司安頓家中寶貝，不必臨時找人照顧或請假，順利兼顧工作與育兒照顧。
