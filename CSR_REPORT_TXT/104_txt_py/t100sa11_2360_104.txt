全球綠色產品的關鍵推手
聯合國氣候變化框架協議簽署方之間的第21次談
判會議 ( Conference of the Parties，簡稱 COP 
21 ) 指出，針對二氧化碳濃度上升所造成的氣候變 
遷，關鍵在於積極推動節能減碳解決方案，包括
如何提升電力能源使用效率，以及開發並使用乾
淨能源。我很榮幸地向大家宣佈，致茂電子高達9
成以上的研發專案，能有效協助客戶達成上述目
標。報告期間我們共締造了27項的具體研究發展
成果，其中25項專案，直接或間接參與了不同產業
的綠色製程或產品實現。進一步來看，22項研發成
果協助客戶提升電力能源使用效率、3項與開發並
使用乾淨能源有關。因此，我們在全球各大產業實
現綠色產品的過程中，扮演舉足輕重的關鍵角色。

成為員工與管理團隊多元化組成的領先企業。讓
我由員工健康關懷的實例中選出一項，向大家說明
員工關懷行動方案。我們已經連續第9年舉辦減重
比賽，並舉辦了超過100場的健康講座，搭配員工
餐廳的健康安心食材，透過誘發員工自主性運動

GRI G4一般與特定標準揭露
編撰一份企業社會責任報告書的核心，是基於重
大性原則來鑑別重大考量面 ( identified material 
aspects ) 。重大考量面意指本公司對於經濟、環境
和社會以及利害關係人的回饋，所彙整的重大影響
議題。依據GRI G4一般標準揭露 ( general stand-
ard disclosures ) 提供2種依循 ( in accordance ) 
選項，包括核心依循選項 ( in accordance-core ) 
與全面依循選項 ( in accordance-comprehensive 
) 。本報告書選擇核心依循選項。接下來，我們著手
規劃 GRI G4 特定標準揭露 ( specific standard 
disclosures )，包括經濟 ( category economic )、
環境 ( category environmental ) 以及社會 ( cat-
egory social ) 。而這三大類別各自再往下展延管
理方針揭露 ( disclosures on management ap-
proach , DMA ) 以及考量面 ( aspects )，被鑑別為
非重大的考量面則不涵蓋在本報告之中。

綠能產業進擊的推手
電力電子測試解決方案，搶先布局綠能商機
我們是世界領先的自動測試設備 ( automatic test 
equipment , ATE ) 自有品牌供應商。擁有9大產品
群：視頻色彩、電力電子、被動元件/電氣安規、半
導體/IC、太陽能、LED、電池、電動車以及光學元
件測試解決方案。其中電力電子測試解決方案更是
全球市佔第一。
面對全球氣候變遷，海內外各國企業前仆後繼的
研發各類低能耗、低污染的綠色產品與服務，例如
太陽能與電動車，他們在推出各式各樣的新能源
與綠能產品前，需要更精確的產品測試服務。看準
綠能產業伴隨而來的絕佳契機，我們把各類測試
服務關鍵技術延伸至潔淨能源，發展出多項技術領
先之產品，包括俗稱LED燈的發光二極體照明燈 
( light-emitting diode lamp ) 之Total Power測
試解決方案，太陽能電池自動檢測效率分級系統、
可回收電能之電池化成系統等，讓本公司及子公司
在測試市場保持領先之優勢，搶先布局全球綠能商
機，更成為綠能產業蓬勃發展的推手之一。
全球超過 600 項專利，國內量測產業第一
致茂亦不斷投入研發資源開發更先進的創新技
術，整合式量測與自動化解決方案 ( Turnkey Test 
& Automation Solutions ）。目前我們在全球已累
計超過600項專利以及100項核心技術。其中包括
台灣370項以上專利，是國內量測產業第一！發明
專利申請也排名全國第73。

我們以高標準的公司治理方針， 
確保董事會有效運作
本公司董事皆以公司與全體股東長期利益為前提，
客觀獨立行使董事職權。我們也深切的了解，越來
越多國內外投資法人與重大利害關係人，皆把公
司治理與是否設立獨立董事，視為投資與否的重
要指標。因此我們持續遵循公司治理 ( Corporate 
Governance ) 原則，由全體股東以投票方式組成
董事會，並於董事會下成立各功能性委員會，以強
化董事會職能。我們以高標準的公司治理方針，確
保董事會有效運作，進而保障股東權益。
董事會職能提升
本報告年度共有5席董事，其中包括2席獨立董事，
獨立董事席次佔比40%。以加強董事會之獨立性
與多元性，發揮策略指導功能。董事會議每季至少
召開1次，2015年共召開6次董事會，平均出席率將
近90%。董事長由黃欽明續任，主責健全公司治理
與主持董事會運作。致茂電子依據公司辦法及《公
開發行公司獨立董事設置及應遵循事項辦法》，董
事之選任應考慮董事會之整體配置，致茂電子董事
會組成整體考量包含：營運判斷與管理能力、會計
及財務分析能力、危機處理能力、產業知識、國際
市場觀、領導能力、決策能力等。
同時，董事會也遵循《上市上櫃公司董事、監察人
進修推行要點》，安排董事至外部進修。2015年董
事進修主題包括：(1)企業誠信風險控管(2)董監事
會運作實務(3)健全公司治理等。
董事會善盡公司治理原則，審核企業經營績效和
討論重要策略議題，包含經濟、環境和社會衝擊、
風險與機會等。董事會之重要決議亦即時公布於
臺灣證券交易所公開資訊觀測站。同時，我們也
公開提供公司章程、董事會議事規則。董事薪酬、
運作情形及對利害關係議案回避之執行情形等資
訊，以供國內外投資者能即時供查詢。
設置薪資報酬委員會，協助評估薪酬水準
本公司依《證券交易法》設立薪資報酬委員會，由2
位獨立董事及1位專家組成，任期3年。2015年共開
會3次，薪資報酬委員會全體成員出席率100%。薪

能，有效將公司的願景、策略目標與員工個人工
作目標結合。藉由(A)2個面向：結果面向、行為
面向，(B)3個階段：期初設定、期中檢討、期末考
核，(C)2個目的：員工發展、人力資源政策之結構精
神，達成上下溝通及組織回饋的目的。同時，考量

行政人員目標不易衡量，仍保留原依特定項目評核

評核結果確實地反映到晉升管理制度、獎酬制度

的運用。全公司員工除事業部級以上主管及外籍同
仁，凡到職三個月試用期滿之員工，皆需進行定期
績效及職涯發展檢視。

醫護室溫馨服務計畫
總公司設立豐富醫護資源，重視同仁隱私之專業
醫務室。並且為拉近環安中心-醫護室與同仁間的
友好關係，讓醫護室不再是同一般的單調冰冷，改
造醫護室成為更貼近人心、溫馨、多樣化且專業的
服務環境。醫護室設立健康期刊區、醫護公佈欄、
配備完善換藥區、量測區、開放室諮詢區、休息
區。醫護室除了提供換藥或看診的健康照顧需求，
更可提供員工減壓談心、小憩片刻與交流健康資
訊。並且也設置女性員工哺乳室。自2012年起，醫
護服務量年人次逐年成長，2015年服務人次2,967
人。

員工薪酬與福利
符合法令退休金
適用《勞動基準法》舊制退休金規定者，依勞動基
準法按月提撥退休準備金，由本公司勞退準備金
監督委員會監督，並以該委員會名義存入，由台灣
銀行負責辦理收支及保管。適用《勞工退休金條
例》退休金新制度者，由公司法依每月負擔不低於
百分之六之勞工退休金提繳率，並按行政院核定
之月提繳工資分級表，提繳儲存於該從業人員在
勞工保險局設立之勞工退休金個人專戶。
周延的保險計劃
我們依法為每位同仁投保勞工保險、全民健康保
險、團體綜合保險。其中團體綜合保險保費由公司
全額負擔，項目包括：定期壽險、傷害保險、重大
燒燙傷、傷害醫療限額險、住院醫療險、癌症醫療
險、職業災害險、重大疾病提前給付 ( 等待期 90 
天 )。
獎金與福利
除了優渥的薪資以外，我們額外提供年終獎金與
紅利獎金，三節、勞動節、生日禮卷。
家庭支援補助金
我們提供多元的員工補助方案。包括結婚、生育、
喪葬、住院補助和急難救助。
優於法令的休假制度
我們除了依法提供同仁休假制度外，更額外提供
同仁優於法規的休假福利，包括1日生日假與5日帶
薪病假。
員工旅遊補助
提供員工國內外自助旅遊補助，實施方式為依據
員工年資，提供自助旅遊同仁現金補助，無旅遊計
畫同仁發放禮卷。並額外舉辦國內旅遊提供員工
報名參加。

淨灘、靜心
2015年9月19日，我們參與荒野保護協會舉辦的淨灘活動，活動為期1天，共計51位公司同仁與眷屬參與。荒
野保護協會自2008年地球日開始清淨海岸，2010年起響應每年9月至10月的國際淨灘行動。我們相信，淨灘
不是解決海洋問題的終點，而是起點，做好垃圾監測及垃圾源頭減量才是保護海洋環境的關鍵。有位同仁
在活動後感性地說到：透過參與這次淨灘活動，讓我覺得不只是單純做環保讓海洋更乾淨。而是在過程中
也讓我靜下心，思考人與環境該如何和諧共處，這的確是很有意義的一場社會參與活動。
