董事長的話
行動實踐永續發展
《2016臺灣產物保險企業社會責任報告書》
為臺灣產物保險第3本報告書，揭露的內容更
加完備至善，這是本公司對企業自身的要求、
對社會大眾的責任，更是做為地球好公民的義
務！為使大眾能掌握更多有關非財務績效的資
訊，未來每年將持續發行報告書，以全面關注
經濟繁榮、社會公益、員工照顧及地球環保的
責任議題，並付諸行動，實踐永續發展之企業
願景。我們不只期許自己成為卓越的企業，更
要成為地球上最盡份的好公民。
秉持「	穩健經營	」與「客戶導向」經營策略
本公司秉持「穩健經營」與「客戶導向」
之經營策略，在風險控管及營運管理方面皆
謹慎實行。本公司資本額36.22億，高居業
界第一，且風險資本適足率達938.87%， 
遠高於法令規定，顯見本公司具有強健的
資本水準及優異償債能力。同時，本年度
持續獲標準普爾公司 ( S&P ) 授予「A-」
展望「穩定」，及中華信用評等公司給予
「twAA」展望「穩定」的優良評等。在
臺灣證券交易所舉辦的「上市櫃企業公司
治理評鑑」已連續3年進入排名前20%， 
不但公司治理成績優於其他產險同業，更於
2016年獲選納入「臺灣公司治理100指數」成
份股。此外，依據金融監督管理委員會公佈
2016年保險公司分級指標評分，本公司排名第
3。顯示本公司在維護股東權益、平等對待股
東、強化董事會結構運作、提升資訊透明度、
落實企業社會責任及保戶權益與理賠服務等面
向，均有高水準之表現。
通過	ISO	27001	認證，接軌國際資安標準
根據金融監督管理委員會的報告，物聯網、社
群、行動、數據分析與雲端等五大趨勢，正逐
漸改變全球的商業模式以及金融版圖。當金融
數位化成為全球銳不可擋的潮流，除了思考
產值與獲益，如何在便捷與安全之間取得平
衡，就成為第一要務。有鑑於此，本公司導入
資訊安全管理制度，並取得 BSI 所核發的 ISO 
27001 資訊安全管理系統 ( Information Se-
curity Management System, ISMS ) 證書，
具體展現本公司在資訊安全管理上的決心。當
然，我們也持續推行並擘劃資訊安全制度，強
化建置數位保險營運實力，使資安防護措施臻
於完善，以因應金融數位化趨勢對資訊安全管
理產生的機會與挑戰。
積極配合主管機關推動政策性保險，成效卓越
為響應金融監督管理委員會推動金融業增進國
內特定民眾的基本保障，及協助建構完善的社
會安全網政策，我們多年來積極配合推廣「微
型保險」及「住宅地震基本保險」，以照顧弱
勢族群並提高國人居住保障。2016 年因為推
廣成效顯著，再次榮獲當年度「住宅地震保險
卓越獎」第一名與「微型保險績效卓著」殊
榮，具體回應政府所呼籲的「善盡社會公益，
補足社會安全網」理念。
2016 TFMI CSR Report  / 5
擴大舉辦產學合作，打造學校、學生與企業三贏
由於深信以人為本是服務業的核心價值，因此我們
致力於培育優秀人才，這不只是企業社會責任的一
環，更是我們的競爭力來源。我們挹注企業資源，
向下扎根，將金融專業與教育志業結合，擴大與多
所學校之保險系所合作，提供(1)課程交流、(2)實習
生計畫、(3)參訪活動，並針對在學成績優異學生提
供獎學金及有薪實習機會，透過多元人才扶植計畫
培育產險菁英，以達到理論與實務的結合。我們期
待藉此計劃，為學生累積實際經驗，提升就業競爭
力，使其發揮所長以回饋企業與社會。這是本公司
履行企業社會責任並展現企業關懷年輕世代的理念
與作為。
取之於社會、用之於社會	
臺灣產物保險與您一起「珍惜此刻、守護未來」
秉持取之於社會、用之於社會的信念，在2015年9
月成立了「財團法人臺灣產物保險文教基金會」。
我們以「珍惜此刻、守護未來」為核心價值，戮力
達致「提升文化品質、培育人才、關懷弱勢、服務
國家社會」等使命，肩負起社會公益的重任。近70
年來，本公司不忘實踐企業社會責任的信念，所以
不論在照顧弱勢、推動遲緩兒特殊教育、宣導青少
年反毒活動，甚至關懷偏郷兒童藝文活動及提倡基
層體育活動等，皆不遺餘力。
未來，本公司仍將秉持「穩健經營」與「客戶導
向」之經營策略，提供保戶全方位的產險服務，以
帶動營運卓越成長、提高客戶忠誠度；同時，也維
持與國際金融趨勢接軌、戮力向下扎根培育人才、
配合政府政策積極作為，並自我期許使本公司成為
國內產險業的標竿企業。更重要的是，未來將透過
基金會的運作，主動關懷弱勢族群，支持學術研究
與體育活動，以善盡企業社會責任。

卓越最佳保險評比 
榮獲最佳永續經營獎
2016 TFMI CSR Report  / 9
身為臺灣歷史最悠久的產物保險公司，除了
致力於發展與時代接軌的產品外，對於風險
管理也同樣重視。本公司資本額36.22億，
高居業界第一，且風險資本適足率 ( Risk-
Based Capital, RBC ) 達938.87%，遠高
於法令規定，顯見本公司具有強健的資本水
準及優異償債能力。展望未來，本公司仍將
秉持「穩健經營」及「客戶導向」之經營政
策，強化公司治理、風險管理及法令遵循，
並持續以「專業、誠信、創新、團隊、關
懷、效率」六大核心價值，對內致力強化公
司治理、員工照護及提升經營績效，對外將
重視客戶關係管理與服務品質，以提升企業
永續發展。
臺灣產物保險獲頒《卓越雜誌》2016卓
越最佳保險評比「最佳永續經營」獎
項，2016年11月15日受邀參加於台北寒
舍艾美酒店盛大舉行的頒獎典禮。當天
由金融監督管理委員會副主委黃天牧授
獎予本公司總經理宋道平，場面隆重。
宋總經理致詞時表示，本公司的成立可
追本溯源自日據時代，以百年歷史守護
寶島產物，獲獎代表本公司在地深耕近
70年的永續經營成果備受肯定。在悠久
的歲月裡，本公司緊緊的守護著每一個
保戶的生命財產安全，同時與行銷通路
及供應商建立緊密的夥伴關係。

報告書概況
這是臺灣產物保險第三次發行 CSR 報告書，
未來我們將每年持續發行此報告書，定期向外
界揭露非財務績效的營運成果，並以行動實
踐永續發展之企業願景。本公司2016 CSR 報
告書再度榮獲英國標準協會 ( British Stand-
ards Institution, BSI ) 臺灣分公司蒲樹盛總經
理頒發 GRI G4 以及 AA1000 雙重國際標準保
證聲明書，並由本公司李泰宏董事長親自出席
授證儀式，彰顯本公司最高管理階層對於企業
社會責任的重視。
報告期間
2016年1月1日至2016年12月31日
上一次報告期間與發佈日期
上一次的報告期間為2015年1月1日至2015年
12月31日，發佈日期為2016年6月30日
報告週期
每年發佈1次
報告書範疇與考量面邊界
本報告書經濟類別的數據來源為2016年企業年
報所刊登的財務報表，環境與社會類別涵蓋範
圍包括總公司懷德大樓3、8、9、11樓，12個分
公司和29個通訊處的整體資訊。
報告期間有關組織規模、結構、所有權或供應
鏈無發生重大變化，對先前報告書中所提供之
任何資訊無進行重編。此外，和先前報告書期
間相比，在範疇與考量面邊界並無顯著改變。
編撰指南
本報告書依據全球報告倡議組織全球永續性
報告指南 ( Global Reporting Initiative, Sus-
tainability Reporting Guidelines Version 4.0 )  
編撰。本報告書選擇核心依循 ( In Accordance 
with Core Option )，針對 GRI G4 一般標

特定標準揭露 ( Specific Standard Disclo-

附有 GRI G4 完整對照表，以供索引各章節
內容。

依據	GRI	G4	揭露一般與特定標準
編撰一份 CSR 報告書的核心，是基於重大性
原則來鑑別重大考量面。重大考量面意指本
公司對於經濟、環境和社會以及利害關係人
的回饋所彙整的重大影響議題。我們著手規
劃 GRI G4 特定標準揭露，包括經濟、環境
以及社會這三大類別，並往下展延管理方針
揭露以及考量面。
鑑別重大考量面與邊界
我們透過鑑別 ( Identification ) 、排序 ( Pri-
oritization )、確證 ( Validation )、檢視 ( Re-
view ) 四大步驟，鑑別重大考量面。首先在
步驟1，我們鑑別相關議題，經過本公司內部
會議，整理一份值得包含在本報告書的議題
清單，又稱相關議題 ( Relevant Topics )。我
們從《GRI G4 永續性報告指南》考量面挑選
相關議題。
接下來執行步驟2排序，同樣依據《GRI G4 永
續性報告指南》界定報告內容三原則，將相
關議題 ( Relevant Topics ) 進行排序。在步
驟1所鑑別的相關議題 ( Relevant Topics )， 
從此階段開始稱為考量面 ( Aspects )。同
時，我們也針對利害關係人蒐集他們關心的
議題，經統計權重分析與內部會議討論後，
獲得議題重大性排序結果。
在步驟3確證的過程中，我們依據 GRI G4 
完整性原則 ( Completeness )，評估已經
鑑別的重大考量面，本報告書經排序後的重
大考量面清單已經過高層決策者核准。而在
鑑別重大考量面與邊界的最後階段，也就是 
步驟 4 檢視，將在本報告發佈後，展開利害
關係人的回饋資訊蒐集，為下一個報告週期
做準備。
視覺化的重大考量面
本報告書所揭露的重大考量面，採用視覺化
的方式呈現，充份反映本公司營運活動所產
生的經濟、環境及社會的顯著衝擊。同時，
透過重大性分佈圖，清楚揭露經濟績效、當
地社區與公司治理，是2016年度最重要的企
業社會責任實踐項目。

針對所議合的利害關係人，我們籌劃嚴謹的
鑑別與選擇方法。首先由臺灣產物保險企業
社會責任跨部門工作小組成員透過內部會
議，激盪出各種可能的利害關係人，接著，
我們依循全球性非營利組織 Account Abil-

ment Standard，簡稱 AA1000 SES 2015 )所
含括的責任、影響力、張力、依賴及不同觀
點這5項標準，針對所有利害關係人執行鑑別
程序與重要性評分，最終獲得優先議合的利
害關係人。包括股東、客戶、供應商、主管
機關、第三方稽核組織、媒體、管理階層與
員工。
針對鑑別出來的利害關係人，我們分別採取
不同的議合形式，包括單向、雙向、面訪、
電話會議、E-mail等，取得利害關係人議合
所提出之關鍵議題。以下是我們回應利害關
係人的標準流程：
• 
各部門定期於內部會議舉行時，反饋外部
利害關係人議題。
• 
各部門主管定期向高階管理階層，匯報利
害關係人關切議題與未來計畫。
• 
本公司企劃部擔任推動企業社會責任之統
籌單位，分析各議題對公司的衝擊，經由
內部持續議合方式，向最高管理階層匯報
議合結果，釐清揭露的重大性議題及內外
部邊界，以及公司如何回應。
• 
舉辦內部稽核，審查對外公開資訊已經擁
有充足的支持性證據。
• 
針對內部議合的過程與結果，皆留下電子
或紙本形式的會議記錄，必要時可作為第
三方查證時的佐證資訊。
• 
確認本年度議題揭露範疇與邊界後，最終
透過報告書或正式新聞稿，公開回應利害
關係人關注事項。
除了上述標準流程之外，我們於官網上設立
利害關係人專區，提供與利害關係人溝通及
聯繫的管道，對於相關的各項議題，如有任
何疑問、建議或甚至是申訴事項，都可藉由

臺灣產物保險成立於1948年，至今已戮力耕
耘近70年，為全國歷史最悠久之產物保險公
司。服務據點遍及全國，也延伸至對岸。本
公司長期秉持「穩健經營」及「客戶導向」
之經營策略，資本額為36.22億元，位居業界
第一名，展現財務績效上的強健資本水準。
臺灣產物保險核心價值——	
專業、誠信、創新、團隊、關懷、效率
「專業、誠信、創新、團隊、關懷、效率」
是我們的企業核心價值。本公司相信，企業
的成功在於提供客戶更有價值的服務，同時
致力於不斷提升專業素養，為美麗的台灣土
地盡一份心力，善盡企業社會責任。
企業標誌的由來
本公司之企業標
誌，是從「臺灣
產物保險」英文
名稱 TAIWAN 的
字 首 「 T 」 為 基
礎發展，並由兩
筆畫重疊組合而
成，象徵本公司
與客戶兩者緊密
結合之寓意，彰
顯本公司與客戶攜手合作、共創未來的經營
宗旨。整體造型亦由臺灣地形衍生而成，企
業標誌圓形部份為「臺灣」之形象，重疊的
三角象徵「中央山脈」，整體輪廓共同傳達
本公司自許為中央山脈一般，守護臺灣的使
命，這更是本企業一貫強調的誠信、關懷正
派經營的企業形象。

個人險種
• 
汽 ( 機 ) 車保險：分為「強制險」與「任意險」。強制險為法令規定須
投保險種，而任意險則依客戶需求自由選擇投保，保險項目包括車體
損失保險、第三人責任保險、竊盜保險、駕駛人傷害保險等保障。
• 
住宅火險及地震險：提供房屋財產安全保障，保險項目包括保險標
的物因火災、竊盜、地震等危險事故所引起的損失，保障客戶的財
產安全及負擔客戶對第三人的賠償責任，對客戶而言是基本不可少
的保障內容。
• 
傷害險：承保被保險人因意外傷害事故所致之醫療、殘廢或身故之
保障範圍。
• 
健康險：健康保險係保障被保險人身體健康狀況的商品，為人身保
險的重要一環，大致可分為「醫療給付」及「定額給付」兩大類。臺
灣產物保險提供「醫療給付」類商品，如住院日額給付保障被保險人
因疾病住院時之補償，「定額給付」如癌症保險或重大疾病保險。
• 
旅遊綜合保險：承保被保險人於旅遊期間因保險事故所造成之旅遊
不便及遭受意外事故之死亡或殘廢與疾病醫療給付。
• 
海外打工渡假遊留學生傷害險：提供海外打工族群保險服務。保險
項目包括身故殘廢、重大燒燙傷、傷害醫療、健康醫療。
• 
高齡長者傷害險：針對50歲以上族群規劃，因為生理性因素例如骨
骼關節、肌肉系統退化以及骨質疏鬆症等，其遭受意外傷害的機率
遠高於其他年齡層，特為年長者構築完善的傷害醫療保障。本公司
提供意外身故、殘廢保險、殘廢看護扶助保險金、傷害住院醫療保
險金日額、燒燙傷病房日額給付等保障。
• 
微型保險商品：提供經濟弱勢者人身保險保障，以補足政府保險或
社會救助機制不足的缺口。經濟弱勢者例如無配偶且全年綜合所得
低於新台幣35萬元者，或是符合《身心障礙者保障法》定義之身心
障礙者。

險公司僅能以「銀行業綜合保險」作為金融機
構轉嫁風險之主要保險，其保障範圍包含員工
之不忠實行為、營業處所之財產、運送中之財
產、票據及有價證券之偽造或變造、偽造通
貨、營業處所及設備之毀損、證券或契據之失
誤致遭受之財物損失為限。對此，臺灣產物保
險掌握市場的脈動與需求，研發新的企業保險
商品，推出了「資訊系統不法行為保險」，以
承保金融機構於實施資訊系統化連線作業後，
為被保險人及其員工以外之人利用資訊系統從
事不法行為所致之財物損失，以保障金融機構
所面臨的新興風險。

降水量參數養殖水產保險問世
為了響應政府推動農業保險，讓水產養殖業者有更
多的保障，臺灣產物保險獲保險局正式核准於3月
30日推出「降水量參數養殖水產保險」。藉此轉嫁
養殖業者風險，並做為養殖業者風險管理之後盾。 
民國98年莫拉克風災在臺灣造成半世紀以來最嚴重
的「八八水災」，全臺漁產損失41億7,380萬元，漁
民設施損失4億8,812萬元，共計46億6,192萬元。
在養殖魚塭部分，受災面積共6,942公頃，損失金
額達到35億5,629萬元。其中又以屏東縣的石斑魚
養殖業損失最為慘重，損失金額約4.3億元。所以
本次首創的「降水量參數養殖水產保險」即是以屏
東縣東港鎮、林邊鄉、佳冬鄉、枋寮鄉等四個陸上
養殖地區之養殖業者先行試辦。 
有別於一般傳統的保險商品，首張參數型天氣保險
最大的特色就是以累積降雨量作為啟動保險契約賠
付要件，無須現場查勘定損，達到簡化程序快速
理賠之效果。養殖業者在保險期間內依據已發生之
累積降雨量數據，依照保險契約約定之條件計算賠
付金額，並於保險金額範圍內對養殖業者負賠付之
責，讓業者迅速得到理賠，將損失程度降至最低。 
「降水量參數養殖水產保險」預期將能協助養殖業
者在最短的時間內完成復育工作及恢復產量，並且
可以安心投資養殖事業、穩定漁貨品質。雖目前僅
以屏東縣作為試辦，未來不排除針對其他地區實際
需求及風險考量，另行開發約定天氣條件之商品，
讓更多農漁民可以藉由保險分散風險，保障農漁民
財產。

風險管理
風險管理委員會，有效控管財務風險
為有效控管各種財務風險，包括保險風險、信
用風險、市場風險、流動性風險、作業風險、
自留風險、天災風險及資產負債配合風險，我
們訂定以下風險管理政策與相關管理準則，以
供各單位遵循。首先，在董事會下設有「風險
管理委員會」，並由獨立董事擔任召集人，執行
董事會授與的風險管理決策，定期檢視公司整
體風險管理機制之發展、建置及執行效能。再
者，風險管理室則負責風險之監控、衡量及評
估等執行層面之事務，並定期提出風險管理相
關報告及壓力測試分析。最後則是即時控管，
臺灣產物保險於各業務單位設立有風險管理人
員以協助單位主管控管可能之風險。
援引國外知名天災模型，因應潛在天災風險
我們具有穩健的財務結構，以承擔客戶所面臨
的天災風險，但對天災所可能造成的風險，仍
具高度的風險管理意識，因此，我們規範嚴謹的
核保流程，援引國外知名天災模型 Risk Man-
agement Solutions® ( RMS ) 以及 AIR World-
wide® ( AIR ) 進行天災風險分析，同時，定期在
各種極端的天災及其他預設情境之下，透過風險
資本適足率 ( Risk Based Capital Ratio, RBC 
Ratio ) 分析，檢測本公司清償能力。
成立個資管理委員會，有效預防保戶個資風險
我們十分重視保戶之個人資料保護，在組織架
構上，成立個資管理委員會作為個人資料風險
控管的最高治理機構，並且每年定期召開兩次
個資管理審查會議，檢視個資維護實施成果，
也制定個資管理政策、規範以及作業程序。倘
若發生個資外洩事件，則依據內部規範《個資
外洩應變流程》啟動標準作業程序。本公司在
2016年度並未發生遭受主管機關裁罰之個資侵
害事件。

透過再保險，轉移鉅額保險責任
從保險產業供應鏈來看，保險公司在承擔
危險的同時，可利用風險轉移，將部分風
險轉移至另一家保險公司，此種降低風險
的方式稱為再保險 ( Reinsurance )。本公司
的再保險供應商包含經營再保險業務之保
險公司 ( Insurance Company ) 及再保險公
司 ( Reinsurance company )。為此，本公司
遵循《保險業辦理再保險分出分入及其他危
險分散機制管理辦法》之相關規定，特制定
《再保險風險管理作業準則》及《再保險往
來機構維護及控管作業要點》，用以規範再保
險人與再保險經紀人選擇。於再保險人選用
方面，必須符合《保險業辦理再保險分出分
入及其他危險分散機制管理辦法》第7條及第
8條之規定，再保人需符合下列標準：
1. 主管機構核准在中華民國境內專營或兼營

營再保險業務之外國保險業。
3. 經國際信用評等機構評等達一定等級以上

工作環境安全
為維護各營業辦公場所服務同仁的安全，本
公司定期配合各地辦公大樓投保公共意外責
任險，並且安裝警民連線系統。同時，也遵
循《消防法》定期辦理消防檢查，更參加每
年兩次的大樓消防安全演練，以確保同仁的
工作環境安全無虞。
職業安全
本公司依據國內《職業安全衛生管理辦法》， 
屬於第三類低度風險事業制定《職業安全衛生
工作守則》及《安全衛生管理制度手冊》，並
依法安排相關單位人員參加甲、乙、丙種職業
安全衛生業務主管教育訓練。結訓後，分派擔
任各分支機構職業安全衛生主管，落實各項
自動檢查制度與教育訓練計畫，期達到「零工
安」之目標。
辦公室空氣品質檢測
為了有效檢測室內空氣品質，防止辦公環境空
氣中二氧化碳濃度太高，導致員工上班精神不
容易集中，總公司依據《勞工作業環境監測實
施辦法》，每年定期檢測環境內的二氧化碳當
量，以維護員工呼吸系統健康。我們委外進行
二氧化碳濃度監測，實施頻率為每年兩次，檢
測結果符合法定容許標準；未來預計納入照明
度檢測。
定期召開勞資會議，同心協力打造勞資和諧
本公司雖無成立工會組織，但仍遵循政府勞
工法令定期舉行勞資會議進行雙向溝通，
更依據《勞動基準法》第83條，制定《勞
資會議實施辦法》與《勞資會議勞工代表選
舉實施計畫》。勞資會議舉行的頻率為每季
一次，必要時得以召開臨時會議，針對促
進勞資合作、勞資關係協調、勞動條件改
善、勞工福利籌劃議題進行雙方討論。此
外，也遵循《勞動基準法》第56條，制定 
《勞工退休準備金監督委員會組織準則》，勞
工退休準備金監督委員會舉行的頻率為每三
個月一次，必要時得以召開臨時會議，會議
內容聚焦在勞工退休準備金提撥數額查核、
資金儲存與支用查核及勞工退休金給付數
額。
透過資方與勞方之密切溝通，顯示本公司在
面臨重大勞工福利議題時，能夠與員工充分
溝通與商討。因為我們深切的了解，促進良
好的勞資關係，將有助於提供良好的工作環
境並降低員工離職率，以維持員工滿意度和
積極性。

84  / 社會公益
臺灣產物保險為國內優質企業之一，秉持「
珍惜此刻、守護未來」的理念，為了讓愛的
種子更加遠播，並以提昇文化品質、培育人
才及關懷弱勢為宗旨，於2015年9月經主管
機關許可成立財團法人臺灣產物保險文教
基金會。未來將有效地整合資源並發揮拋磚
引玉的精神，與本公司母公司相輔相成達到
一加一大於二的效果，持續以愛與關懷投入
公益、獎勵學術研究、提倡藝文及體育、獎
助清寒優秀學生，使企業經營融合社會的脈
動，為社會盡最大的心力。
此外，本公司所屬的領航集團也積極參與社
會公益，2006年成立財團法人領航社會福利
慈善事業基金會，作為企業社會責任的專責
組織。領航基金會捐贈類別包括(1)清寒孩童
獎學金、(2)醫療與急難救助、(3)特殊教育
救濟、(4)其他。以清寒孩童獎學金捐贈專案
而言，領航集團李文勇董事長出生於新北市
板橋區，基於回饋鄉里的使命感，與新北市
政府社會局，板橋區公所社教福利課合作，
設立清寒孩童獎學金捐贈專案，針對板橋區
領有低收入戶證明的公私立國小及國中之在
學學生，經由主動遞件申請以及公平程序遴
選後，每位學童每學年可獲得3,000～5,000
元獎學金。領航集團自2007年開始開始贊
助，累計受益人數1,930位。

臺灣產物保險有感於校園毒品氾濫及青少年
染毒的情形日益嚴重，且年齡層逐漸下降，
為喚起更多人對於青少年反毒議題的重視，
自2015年起贊助紙風車發起的「拯救浮士德
計畫」。
浮士德源自於歐洲中世紀文學作品裡的虛構
角色，為了取得至高無上的權力，浮士德不
惜出賣自己的靈魂，向魔鬼作出交易。
對於那些不慎接觸毒品而難以自拔的青少年
而言，他們就如同劇中的國中生小得一般，
因為被老師忽視、受同學排擠霸凌，而讓魔
鬼以金錢、毒品誘惑，渾然不知吸毒已讓自
己身陷險境。紙風車劇團透過寫實的劇情，
在70分鐘內以3段生動的故事，以毒品就像惡
魔的劇本主軸，讓學生觀賞後能了解毒品的
可怕危害，並提升對於毒品及藥物濫用的認
知及態度。

造健康校園。
