董事長的話
華新麗華從 2014 年起，參考 GRI -G4 指標編纂企業社會責任報告書，
藉此以誠信、透明的態度，檢視及揭露我們在 CSR 工作上的當責性與
執行成效，並也確切溝通與回應所有利害關係人的關切與期待。隔年我
們更進一步透過「企業社會責任委員會」整合性權責組織，將永續經營
的企業文化與績效目標，與「公司治理」、「環境保護」與「社會共融」
三個行動面向緊密結合，貫徹於日常營運之中，導引我們將經營的角度
逐步轉為長期策略的思考，進而以更加謹慎、優於法規的思維去探究企
業永續的議題。
2016 年國際金融市場持續紛擾、總體貿易需求減緩等不利因素衝擊下，
全球經濟復甦不如預期。然而公司持續落實「聚焦核心」策略，強化營
運體質、提升管理績效，力求在快速變化的產業環境中，維持競爭力並
確保穩定獲利。這一年我們除了努力經營好現有事業外，也同時啟動了
智能化相關的規劃與建設，因應製造服務高值化的發展趨勢，打造永續
經營的成長實力。

1.1 事業發展
1.1.1 華新發展介紹
華新麗華股份有限公司創立於 1966 年，以資本額三千萬臺幣、生產電
線電纜起步，目前華新麗華已是大中華區電線電纜及不銹鋼產業領導廠
商，同時成功跨足商貿地產與科技產業投資的國際化企業。
1972 年於臺灣證券交易所掛牌上市，所生產的銅線材、電力電纜、通
信線纜、不銹鋼材，廣泛運用於電力傳輸、電信網路、交通運輸、工業
生產等基礎建設。2016 年合併營業額約新臺幣 1,422 億元，擁有員工
近五千人。
華新麗華在九○年代初即積極展開以中國大陸為成長重心的營運佈局，
至今已於全球設立十餘座生產與銷售據點。
旗下核心事業中，電線電纜事業包含銅材、電力與通信電纜。電力電纜
與通信線纜產品線完整，深耕大中華電力和電信需求。不銹鋼事業生產
不銹鋼鋼胚、盤元、直棒、無縫鋼管等，為亞洲專業之長條類不銹鋼廠。
商貿地產事業聚焦兩岸經貿商機，同時以累積三十多年房地產開發管理
經驗，在中國大陸城市推動具規模之地產物業開發。
華新麗華自 1980 年開始轉投資半導體、電子零組件、印刷電路板、光
電與金融投資等產業，成立包括華邦電子、華新科技、瀚宇博德、瀚宇
彩晶等公司，藉由發展半導體及先進材料產業的經驗及成果，作為華新
麗華佈局新興科技領域的基礎。
華新麗華始終掌握趨勢，以前瞻性的佈局策略追求企業創新成長；並持
續與國內外知名產學界合作，進行自主技術之研究，依據市場及客戶需
求開發新產品與新業務。華新麗華秉持一貫對品質的嚴謹要求，以及快
速整合的服務，成為客戶的最佳伙伴。在營收持續成長獲利的同時，對
社會與環境做出具體貢獻，與社會共同成長是我們努力的目標。落實於
日常營運之中，藉以驅動我們在經濟、社會、環境各個發展面向都能有
穩健持續的進步，為永續企業經營打下穩固的根基。
穩健經營
華新麗華秉持誠信經營理念，一步一腳印耕耘核心事業，做好風險管
控。在致力於穩定獲利的同時，持續將企業社會責任之經濟、社會及環
境各面向所關注的議題，透過公司治理體系，從目標設定、檢視到修正
的運作過程，內化於日常營運當中；從持續推動綠色生產、帶動員工積
極參與環保行動及社會志工，凝聚全員對社會公民責任的共識，追求企
業永續發展目標。

不銹鋼事業
商貿地產事業
不銹鋼事業生產不銹鋼鋼胚、盤元、直棒、鋼捲、無縫鋼管、鋼纜等。
擁有年產 70 萬噸（扁）鋼胚之規模，其中不銹鋼長條類鋼材產量在亞
洲首屈一指，上下游垂直整合佈局，具備從煉鋼、軋製、退火、酸洗、
削皮、冷抽一貫化作業，提供高品質服務，交期迅速與客製化產品，在
生產規模、品質與技術各方面皆具競爭優勢。
2016 年下半年受美國大選結果、菲律賓禁止原礦出口等影響，大宗商
品價格緩步走升。其中 LME 鎳價從 6 月底約 8,300 USD/MT 逐漸上漲，
至年底約 11,000 USD/MT，上漲幅度逾 30%。在市場資金較為充裕下，
下游備庫存意願增加，因而帶動需求。雖全球政經局勢不穩定，中國內
需市場成長趨向緩慢，但在中國因環保政策而控管產能投放，市場供需
亦趨於平衡。
展望 2017 年，國際政壇的不確定性，市場仍在觀察美國新政府政策走
向、英國脫歐等事件對經濟情勢之影響。目前鋼鐵產業市況仍處於供
過於求，各國競相利用反傾銷 / 反補貼或提高關稅等貿易壁壘，恐將成
為業者維持及開拓市場的阻礙。然而中國將延續 2016 年的環保去產能
政策，打壓過剩產能，預計會有更多類似寶鋼與武鋼合併的鋼廠整併重
組案，將可適度減輕產業過度競爭之壓力。雖中國經濟處於轉型升級過
程，GDP 增速不如過去幾年強勁，但在加大供給側結構改革力道的環境
下，而本公司屬符合 < 鋼鐵行業規範條件 > 的鋼廠，憑藉著穩定的產品
品質與具競爭力的成本優勢下，勢必能在政策迫使鋼鐵行業供應端重新
洗牌的趨勢下，進一步深化中國不銹鋼市場經營與品牌價值拓展。而美
國新政引導製造業回流，可望啟動基礎建設的需求，加上歐盟取消對臺
南京華新城為本公司位於南京市之地產開發項目，包括辦公樓，住宅及
商業等；建築面積達 100 萬平方米，位於南京市河西地區二號地鐵與十
號地鐵的交會處，地理位置優異，具開發潛力。有鑑於南京房市成交均
價增長幅度過於迅速，2016 年 9 月較 1 月上漲高達 27%，因而南京政
府進行“限購限貸” 調控政策以抑制房價。惟華新城所處的河西中部土
地已基本開發完畢，新房源少，且地段位置極佳、客源充足，故此次限
購政策對華新城項目銷售影響不大。就長期而言，平穩、健康的市場對
華新地產項目的銷售與運營則更為有利。
展望未來，隨著河西新城區的最終成熟，地段優勢將更加突顯，住宅與
辦公產品的稀缺性進一步增強，商業客群規模更大、消費總量也更為穩
定，華新城“品質標竿”的品牌影響力與美譽度也更加深入人心。
扣件反傾銷稅率，可拉抬臺灣的產業群聚 ( 盤元 / 扣件 / 線材 ) 效應，
在預見下游客戶市場仍可持續成長下，" 華新 " 因應客戶需求亦啟動設
備更新等資本支出、提升效率，增加客戶對 " 華新 " 的黏著度。同時加
強高附加價值鋼種和優化產品組合的銷售策略，追求質、量同步成長，
朝極優化發展，使 " 華新 " 長條類產品將取得更佳的市場地位。

所謂「誠信經營」公司文化，乃是要求公司

動當地相關法令規章。所有公司成員不得一

2016 年兆豐金事件，法遵議題再度獲得重視，
特別是金控公司。本公司為傳統製造產業，
主要營運法律遵循風險，仍在製造有關的勞
動、環保法令乃至衝突礦產的拒用；銷售部
分則包含行業主管機關對於消費者安全、健
康等權利的保護以及公平交易法等。財會方
面，主要為租稅法令、甚至各國的追稅規定、
乃至反洗錢等相關規範。上市櫃公司則受制
於公司法、證交法乃至相關公司治理與企業
社會責任規範。
公司必須密切注意任何可能會對公司業務及
財務有重大影響的國內外政策與法令動向。

法規，隨時關注法規動向：

4.2.3 供應商分享
50 年來，本公司視供應商為長期合作的夥伴關係，並倡導與供應商一
同建立永續供應鏈，經由充分溝通並互相合作以追求雙贏。這些來自合
供應商專欄小故事
高智工業有限公司 ( 新莊 )
早期包裝用鐵軸都只有標準規格，而每個規
格能裝載線纜的種類尺寸有限，造成包裝軸
使用範圍受限。因此回收鐵軸時必須每個規
格都準備，但是某些冷門規格鐵軸放置久
了，因日曬雨淋而造成損壞必需報廢，間接
形成對資源的浪費。
因此為了改善此狀況，我們與高智共同研究
較為特殊可容納更多線種尺寸的鐵軸，經多
次廠內測試及與客戶溝通試行，現已漸普及
在市場上且後續又開發幾款特殊軸來替代原
眾多尺寸的鐵軸，而此幾款特殊軸也因回收
使用率高，進而對環境資源的節省做出更多
貢獻。高智與華新在鐵軸的合作上已數十個
年頭，從早期的標準鐵軸到後來的特殊規格
鐵軸，就猶如華新的產品及服務越來越多元
化一般，高智也因與華新的共同研究開發而
逐漸成長，讓高智的產品也能有多樣化的發
展。期望未來高智與華新的合作能夠讓雙方
共同在企業及環境的永續發展上更向前邁進
一大步。
Magnesita德商鎂西塔礦業有限公司(鹽水)
Magnesita 耐火材料公司自 1993 年華新麗
華建廠初期，即參與 MRP 轉爐及 VOD 盛桶
設計，並於 1994 年華新投產之後，MRP 及
VOD 開始使用 Magnesita 產品。
使用初期，MRP 壽命為 70-80 回，VOD 為
8-10 回，經雙方共同討論煉鋼製程及造渣調
整，逐一去除煉鋼程瓶頸並變更耐火材料設
計，研發新材質應用，再透過國外技師定期
拜訪互相交流探討，國內技師每周跟催配合
耐火材料管理，隨時提供雙向資訊，以調整
設計和材質。迄 2016 年止，MRP 壽命已提
昇至 500-600 回，VOD 大幅改善達到 38 回，
Magnesita每年銷售量也達到3,500噸以上，
此雙向合作，不僅讓使用者成本降低、產量
增加、品質改善，供應商亦增加了銷售量，
實為雙贏之最佳闡釋。 
台灣住友商事股份有限公司(重要資材管理部)
住友商事跟華新麗華之間的生意往來，從第
一批銅板交易開始已經過了三十個年頭。從
住友金屬礦山的銅板，到馬來西亞的鋁錠、
南非的鎳粒以及日本金線的供應等，在在地
顯示出兩家公司的關係非常密切。而歷屆的
日本住友商事總公司的會長、社長到臺灣拜
訪時每每必定造訪華新，其重要性可見一
班。中村社長、宮原 ( 前會長 )、岡素之 ( 前
會長 ) 也都對華新兩代間的兄友弟恭、視野
宏寬，見解獨到十分稱許。
在台灣住友重新劃分歸屬在東亞這個經濟區
塊後，未來台灣住友與中國住友間的互動勢
必會更加頻繁密切的情況下，本人安東也期
待透過華新麗華與中國當地的深耕基礎，能
夠開發出更多的合作機會，讓雙方關係更加
緊密而邁向下一個新的三十年里程。
作盟友與夥伴企業源源不絕的力量正是華新麗華茁壯的最大動能。

退休金制度
本公司遵循各地相關退休法規及制度，保障員工退休權益，使同仁皆能
於退休後享有安心的生活保障。
臺灣地區員工得選擇適用勞動基準法有關退休金規定（舊制），或適用
該條例之退休金制度（新制）。公司每年按每位具舊制退休金年資者之
2% 月薪資，按月提存至臺灣銀行舊制退休準備金帳戶，及提列『應計
退休金負債（準備）』費用；配合 2015 年《勞動基準法》第 56 條修正
通過，本公司依法於年底檢視勞工退休準備金專戶餘額是否足夠支付未
來一年內成就退休條件之員工所需退休金，並於次年 3 月底前一次補足
差額，以強化員工退休金之保障。對於適用及選擇新制退休金之員工，
公司按月依其薪資 6% 提撥至勞工保險局之員工個人退休金專戶，員工
另可依個人意願選擇提撥 0% ～ 6% 不等之退休金，提存至個人退休金
專戶。
大陸地區依員工薪資總額 13% ～ 21% 按月提存養老保險金，提供年老
退休後的保障。
社會保險及員工團體保險
華新提供各地區員工符合當地法令之社會保險，以保障員工基本權益。

1. 總損失工時 ( 日 )：當年度所有工傷之總缺勤天數 ( 天 )
2. 職災人數：當年度所有工傷人次
3. 年死亡千人率：報告期間因公死亡人數 / 總勞工人數 *1,000 ( 每年每一千人中之死亡率 )
4. 失能傷害嚴重率 (SR)= ( 失能傷害人日數 X1,000,000) / 總經歷工時  ( 相當於 5 倍 LDR- 誤工率 )
5. 失能傷害頻率 (FR)=( 失能傷害人次數 X1,000,000) / 總經歷工時     ( 相當於 5 倍 IR- 工傷率 )
6. 損失天數比率 (LDR) = 工作中因病 ( 傷 ) 總缺勤天數 ( 天 ) / 總經歷工時 ( 小時 )  X 200,000
7. 缺勤率 (AR) = 工作中事病工傷總缺勤天數 ( 天 ) / 總經歷日數 ( 天 )  X 200,000
8. 職業病率 (ODR)= 職業疾病總數 ( 件 ) X 200,000 / 總經歷工時 ( 小時 )
9. 綜合傷害傷害指數 (FSI)= √ ( (FRxSR/1000)= 即綜合 F.R. 與 S.R. 兩種比率所得的指數，為 F.R. 與 S.R. 乘積除以
1,000 的平方根。

5.3.6 承攬商安全管理
承攬商管理與稽核
本公司除自身盡力於企業社會責任外，也延伸拓展至對承攬商的要求，以往對承攬商之管理，從
產品本身及發包專案的角度去要求的稽核方式已明顯不足，本公司針對供應商及承攬商的稽核及
評選以更積極的態度從環境、社會及經濟等各個面向充分考量，進行妥善有效的管理，並掌握供
應鍵顯著或潛在風險。
除上所述，依本公司行業之特性，亦特別密切關注承攬商下列幾項之規範，以確保同仁之職場安全：
◢ 定義高風險作業進行重點管制。
◢ 要求承攬商依法提供專業技術人員資格證書。
◢ 承攬商廠施工前須完成開工會議、協議組織會議。
◢ 定期查核承攬商施工情況。
・本公司致力於提升承攬商施工品質、工作安全及素養，於 2016 年共執行 634 場開工前會議、
28 場協議組織會議；並於會議過程中提及施工前應注意事項、施工中危害告知等相關事項。
・為要求施工之安全，本公司不定期派遣人員至工區執行查核情況，如有發現作業危害點，除
立即開立改善單 (2016 年共開立 59 件罰款單 )，並要求即期改善；若情節重大或屢勸不聽者，
則給予立即停工改善或開立相關罰款通知單要求改善，落實管理制度，提升承攬商工作安全
及工程品質。

6.1 氣候變遷與節能減碳
隨著全球低碳經濟發展趨勢，企業必將在環境風險管理上扮演日益重要
的角色，氣候變遷所造成的極端氣候，例如風災、水患、乾旱與暴風雪
等災難性氣候，發生的頻率將愈來愈高，不僅會對社會與環境帶來嚴重
影響，也會造成巨大的經濟損失，衝擊企業及其供應鏈。
氣候變遷的因應措施，已從早期的政府責任擴展為全民參與。企業具有
實質影響力，採取的作為將具備示範性與帶動性，故面對氣候變遷帶來
的諸多風險，華新麗華透過不斷地學習、自省、改善與創新，從溫室氣
體盤查、能源管控及節能減碳推行，並且借鏡標竿企業的作為，也審視
自身核心價值與競爭力，期望成為業界領導楷模為目標。

6.1.2 溫室氣體排放盤查
對抗氣候變遷與全球暖化，溫室氣體減量是重要手段，盤查則可提供減
量依據。經由盤查結果可以訂出減量目標與優先順序，讓後續的減量過
程更有效率，並且可以藉以確認減量成果。
華新麗華公司自 2013 年開始，每年確認其溫室氣體排放量，做為自身
節能減碳的第一步，並且積極地將碳盤查與減量，並且建置『安環資訊
平台 - 溫室氣體盤查與計算產品碳盤查的能力』每年所有臺灣廠區及海
外廠區都必須完成其前一年該廠區之範疇一與範疇二之溫室氣體盤查，
並且積極推廣須通過外部驗證機構符合 ISO 14064-1 標準之查驗。目前
鹽水廠已通過 ISO 14064-1 標準之查驗，預計 2017 年臺中廠也將取得
外部驗證機構符合 ISO 14064-1 標準之查驗，同時，公司亦建立了內部
環保、安全與衛生資訊系統，各廠區須定期將盤查結果登錄於此系統
中。公司綜合這些努力建立了整體產品碳盤查及碳管理能力，增加產品
的國際競爭優勢。

6.3 綠色營運
積極致力於追求環境平衡並減輕環境負荷是本公司的信念，故除在生產
製造過程持續進行環保設備投資以降低對環境影響外，更不斷致力於製
程精進、製程簡化、製程減少及製程調整等，以減少資源使用、降低對
環淨污染，並達到節能減碳作用。此外，更積極開發與擴大綠色產品的
供應，成為帶動綠色供應鏈的重要推手，以為節能減碳、減緩地球暖化
貢獻心力。
華新麗華的綠色營運從四大領域進行優化與改善，以期能永續發展 :
◢ 綠色製程開發
◢ 綠色產品
◢ 產品有害物質管制
◢ 副產物減量回收再利用
2016 年投入下列綠色製程開發，將更進一步降低對環淨污染，並達到
節能減碳及節省資源等作用。
◢ 開發煉鋼爐渣改質處理與再利用技術，以減少爐渣對環境衝擊。
◢ 開發煉鋼爐渣減量製程，以減少爐渣對環境衝擊及節省資源。
◢ 引進純氧燃燒技術，以減少燃料耗用並可降低 NOx 排放。
◢ 熱處理製程優化，以降低燃料耗用，節能減碳。
◢ 酸洗製程減少。使用機械除銹取代部份酸洗製程，來減少化學藥劑
使用，減少對環境污染及節省資源。
◢ 推動進一步提升酸洗廢硫酸回收與再生設備之新增，以減少硫酸液
使用，節省資源。
◢ 臺中廠酸洗製程新增廢混酸處理與再生設備，進一步提昇廢混酸處
理能力及再生量，可減少酸液使用，節省資源。
◢ 推動新增酸洗設備進一步減少排放氣體中 NOx 含量及放流水中硝酸
鹽氮含量，減少對環境污染。
◢ 方棒製程精進。調整方棒冷抽製程進而減少中間冷抽與退火次數，
降低了模具與退火燃料耗用，節能減碳。

6.3.2 綠色產品
綠色環保型產品 - 不銹鋼
不銹鋼由於具有優良耐蝕性及優良耐熱性，於嚴苛環境下使用能減少設
備失效而造成停機，延長使用壽命，且可 100% 回收、再生。本公司不
銹鋼生產之原料約有 50% 以上使用回收廢不銹鋼，得以減少廢棄物而
造成對地球環境之負擔，故不銹鋼可視為綠色環保型產品。此外，不銹
鋼具較低生命週期成本的特色，亦即，雖不銹鋼添加了較貴重元素 - 鉻、
鎳，雖造成生產成本較一般鋼材為高，但若以生命週期成本 (Life Cycle 
Cost) 來計算，則由於不銹鋼具有遠較一般鋼材更優良耐蝕性及使用壽
命，而使不銹鋼實際成本較一般鋼材為低廉 ；下列圖表為引用自 ISSF
資料 ( Module 07 Sustainability.pptx)- 不銹鋼 LCC(Life Cycle Cost，生
命週期成本 ) 案例 - 屋頂用料，其為針對不銹鋼之 LCC 所做案例之一；
由圖表可看出，不銹鋼之 Life Cycle Cost 較碳鋼 (carbon steel) 為低。
