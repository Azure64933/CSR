智邦科技2016企業社會責任報告書， 
依循GRI G4 以及AA1000 雙國際標準編撰
為了實踐永續經營的企業願景，並讓抽象的議題更
加具體明確，智邦科技於今年首度發行企業社會責
任報告書，向外界揭露非財務績效之外的營運成
果。未來，我們將每年發行此報告，持續不懈地朝
向永續發展的目標邁進。
報告期間
2016	 CSR	 Report	 的資訊揭露期間為年2016年1月1
日至2016年12月31日。
本報告期間關於組織規模、結構、所有權或供應鏈
等皆無重大變化。
報告週期
報告發布週期為每年發布1次。
報告書範疇與考量面邊界
本報告書經濟類別的數據來源為2016年企業年報所
刊登的合併財務報表，在環境類別和社會類別的揭
露，則以台灣新竹營運總部為揭露邊界。
編撰指南
本報告書的編撰依據為全球報告倡議組織全球永
續性報告指南	(	Global	Reporting	Initiative,	Sus-
tainability	Reporting	Guidelines	Version	4.0，
簡稱	GRI	G4	)	，透過核心依循	(	in	accord-
ance	with	core	option	)	的準則，針對	GRI	G4	一
般標準揭露	(	General	Standard	Disclosures	)	與特
定標準揭露	(	Specific	Standard	Disclosures	)	，呈
現本公司2016年度的永續議題與資訊揭露；同時提
供	GRI	G4	之完整對照表，以供索引各章節內容。
查證標準與結果
為提高本報告書之可信度，智邦採用全球性非營利
組織	 AccountAbility	 所制定的	 AA1000	 保證標準	
2008年版(AA1000	Assurance	Standard)	的第1應用
類型(Type	1)、高度保證等級(High	Assurance)作為
查證標準(Assurance	standard)，來針對報告書所揭

之縝密完備，智邦未來預計邀請第三方查證機構，
協助我們進行外部查證。

董事長的話
「信任」「熱情」「勇敢」「合作」「完美」是智
邦集團在科技與人文之間一直不變與深信的核心價
值，人是公司最大的資產。智邦的企業識別標誌	 「
集智之樹」所代表的是集結工作夥伴及客戶夥伴們
的智與慧，藉由互動互助相互激盪，發揮出群策群
力最大的力量，創造豐碩的成果。
在網路通信科技技術不斷變化的過程中，我們始終
相信企業的成長與獲利必須架構在讓人們更幸福、
讓社會更優質的經營理念上。對內的員工照顧與對
外的社會公益與文化藝術一直是公司成立以來默默
進行的活動，智邦科技於2011年獲得新加坡CMO	
Asia	評選為亞洲Best	CSR	Practice	Award。
隨著公司拓展成擁有數個關係企業的智邦集團，強
烈的使命感與社會責任感讓智邦集團積極投入回饋
社會的活動，並將環保意識帶入從設計到製造的全
部流程。展望未來，智邦的使命：
•	
持續用我們的熱誠提供IP-centric的優質解決方案
•	
保持全球開放資訊科技基礎架構(Open	IT	Infra-
structure)的領航者地位
•	
致力於運用我們的科技回饋社會
在2016年智邦邁入第29年，我們把Corporate	 So-
cial	Responsibility	企業社會責任列入公司年度自我
檢視的目標，在業績與獲利成長之上，我們要求公
司治理，平衡與利害關係人的利益，著重環境永續
的生產，要求管理供應鏈，融合多樣性與包容性，
與回饋社會。
維持綠能環境的可持續性
當世界正在改變，我們對於企業責任的承諾將一本
初衷，將共好使命整合日常業務，為智邦創造永續
營運價值。我們將持續運用核心技術，來協助解
決氣候變遷和能源效率問題。本公司取得國際ISO	
14001	
認證，建立環境管理系統，落實推動並整合
至內部管理體系與生產活動，包括製程、產品與環
境間的相互關係。
全面加強供應鏈管理　擴大綠色供應鏈管理方針
制定全新供應商管理政策，加強綠色供應鏈管理方
針。智邦所使用之零部件、模組、材料等產品，
均完全符合《智邦產品危害物質準則管理程序》
(QPC-CE05)中危害物質管制清單的規範，而且限
用物質管制不僅僅侷限於EU	 RoHS危害物質限用指
令所管制的6項物質，更包含了國際環保法規與國際
客戶的管制項目。我們亦建置「綠色供應鏈資訊管
理平台」，由源頭設計即導入綠色材料，大幅提昇
綠色產品的可靠度與相關驗證時程。
融合多樣性與包容性
員工是智邦最重視的資產，我們致力於提供員工優
質的工作，包含具競爭力之薪資福利條件、完善升
遷管道及安全的環境，並塑造可以持續學習、致力
創新並充滿樂趣的工作氛圍，以吸引並留任優秀人
才，確保人力資源結構的多元化與包容性。
持續推動社會公益與人文藝術活動　擴大員工參與
自1999年成立智邦公益館以來，透過智邦公益館
平台持續照顧社會弱勢，關懷弱勢學童，並鼓勵員
工參與善行，讓愛的種子在同仁內心發芽，不斷茁
壯。2000年以來，智邦藝術基金會持續推廣文化藝
術教育，近年來透過與社區合作活化古蹟建築，推
廣社區民眾參與，成效日顯。
一路走來，智邦堅持正直與誠信經營的企業理念。
「Making	Partnership	Work」展現智邦對合作夥伴
的承諾，也是我們為建立長久的夥伴關係秉持的最
高信念，帶給所有人最大的利益。
隨著我們持續向上提升的營運績效，我們將致力於
揭露更多非財務面向的可持續性報告。展望未來，
我們會繼續秉持正直與誠信的企業理念，戮力於企
業與社會的永續發展，並透過更多透明化的企業社
會責任資訊釋出，歡迎各界利害關係人持續關注我
們的永續經營理念，也竭誠歡迎您提出反饋，督促
我們能永續茁壯。
智邦科技董事長	李炎松

公司概況

智邦科技1988年成立，總部位於台灣新竹科學園區，致力於全方位乙太網路和無
線設備之研發、設計和製造，藉由多元完整的產品線、優異的通訊技術研發技術以
及與全球網通產業上下游供應鏈緊密的夥伴關係，為頂尖網通、資訊、電信業者提
供多元化網路通訊設備及解決方案，於台灣、美國、中國等設有研發和銷售中心，
全球超過3,300位員工，並於1995年於台灣公開發行上市(股票代碼：2345)。截至
2016年年底，智邦台灣地區員工人數為1,286人，隨著市場發展與產能上升，員工
人數相較2015年度上升13.5%。
在行動、寬頻、整合、雲端虛擬化的
全球趨勢下，持續開發下一世代網路
架構(Next	Generation	Network)，藉
由建立單一IP通訊平台以整合串連所
有的資訊與服務。從消費末端到電信
局端，分別提供了整合數據、影音、
行動等軟硬體設備，產品線涵蓋數據
中心交換器、城域和企業級交換器、
企業無線接入、寬頻擷取和閘道器、
智慧傳感器等，為全球頂尖客戶持續
不斷提供快速且完整的服務，達成行
動寬頻和數位生活的遠景。

智邦科技企業社會責任政策
•	
遵守法律法規：遵守當地政府相關勞動、環保、
安全衛生等法規要求。
•	
確保就業自由：確保所有工作都是自願性的，不
使用壓迫、強制性條件雇傭員工。
•	
實行人道待遇與禁止非法歧視：不使用童工，不
以體罰、虐待、強迫性方式對待員工，在聘用、
報酬、培訓機會、升遣、解職或退休等事項上，
不會有種族、民族或社會出身、社會	
階層、血
統、宗教、身體殘疾、性別、性取向、家庭責
任、婚姻狀況、工會會員、政治見解、年齡的歧
視。
•	
提供合理薪資與福利：嚴格遵守當地政府與勞工
薪資福利相關之法規，並公開及明確地對員工實
施公司各項管理。
•	
智慧財產權保護：保護並尊重公司及他人之智慧
財產權利。就技術、專業技能及其相關檔案與資
料之使用須以保護智慧財產權之方式為之。
•	
透明化：建立對內、外的相關溝通管道。對內，
公司應鼓勵員工與管理階層間之直接溝通與分
享；對外，公司將歡迎任何透過專用信箱之訊息
交流。
•	
堅守誠信經營：就公司營運及員工操守之管理將
採取最高廉潔道德標準。禁止任何形式的賄賂、
貪污、欺騙等不正當行為。
•	
經營並推廣社會參與：公司將積極參與社會活
動，鼓勵公司業務夥伴共同推動並善盡企業社會
責任。
•	
無衝突礦產政策：對於近年來，來自剛果民主共
和國衝突礦區之礦物，礦物類別包含鈮、鉭、鐵
礦、錫石、黑鎢礦與黃金等。而這些礦物被提煉
成鉭(Ta)、錫(Sn)、鎢	 (W)、金(Au)等，造成社
會、環境、人權惡化等問題，本公司將持續採取
無衝突金屬採購政策，支持EICC與GeSI在衝突
金屬上的策略與做法，並已採用EICC-GeSI聯
合工作組發佈之衝突礦產報告範本管理工具，
調查本公司供應商是否落實上述政策。更多關
於EICC的資訊，請造訪以下網路連結：http://
www.eicc.info。

鑑別重大考量面與邊界
本報告書之重大考量面，是透過鑑別	(	Identification	)	、排序	(	Prioritization	)	、確證	(	Validation	)	、檢視	(	
Review	)等	4大步驟鑑別而來。
步驟1：鑑別
首先，由專責企業社會責任的單位召開內部會議，從	GRI	G4	考量面	(	Aspects	)

客觀獨立行使董事職權。我們也深切的了解，越來
越多國內外投資法人與重大利害關係人，皆把公司
治理與是否設立獨立董事，視為投資與否的重要指
標。因此我們持續遵循公司治理(Corporate	
Gov-
ernance)原則，由全體股東以投票方式組成董事
會，並於董事會下成立各功能性委員會，以強化董
事會職能。我們以高標準的公司治理方針，確保董
事會有效運作，進而保障股東權益。
董事會職能提升
本報告年度共有9席董事，其中包括2席獨立董事，
獨立董事席次佔比22%。以加強董事會之獨立性與
多元性，發揮策略指導功能。董事會議每季至少召
開1次，2016年共召開6次董事會，平均出席率將近
82%。董事長由李炎松擔任，主責健全公司治理與
主持董事會運作。智邦科技依據《公開發行公司獨
立董事設置及應遵循事項辦法》，明訂董事之選任
應考慮董事會之整體配置，智邦科技董事會組成整
體考量包含：營運判斷與管理能力、會計及財務分
析能力、危機處理能力、產業知識、國際市場觀、
領導能力、決策能力等。
同時，董事會也遵循《上市上櫃公司董事、監察人

意度問卷，並按照客戶建議持續改善。

在定義的時間內進行妥善的處置與回覆，避免影響
客戶的滿意度與公司聲譽。針對客戶提出的軟體問
題，會確認已登錄公司的PTR系統，進行對策有效
性與時效性的追蹤，至客戶滿意結案為止。
為了讓客戶滿意度提升，智邦設置材料監控系統，

空污管理政策
近年來空氣品質日益惡化，逐漸加深對人類健
康的影響程度，各種排放源之粒狀污染物即是元
兇，PM2.5在近年來已成為粒狀污染物排放之重
要指標之一。智邦科技針對製程有機溶劑之使用
所產生之廢氣，經廠內污染防制設備前處理後，
排放於大氣中，並依循《空氣污染防制法》，進
行固定汙染源空污費暨排放量申報，經由固定汙
染源空污費暨排放量申報整合管理系統計算後，
排放量低於排放門檻標準。
•	
廢污水管理政策
本公司依循《水污染防治法》，鑑別生活污水與
製程廢水。我們於製程中沒有產出廢水，僅有辦
公場所產出之生活污水，而生活污水經廠內前處
理後，放流水氨氮比均低於排放標準75mg/L，而
直接排入工業園區污水處理系統。每半年執行放
流水水質檢測，藉以監控放流水排放前水質符合
標準。
•	
廢棄物管理政策
智邦科技依循《廢棄物清理法》鑑別與妥善清理
廢棄物，所有廢棄物皆經由政府立案核准之廢棄
物清除、處理公司清運、處理及再利用。

溫室氣體減量目標，每年減少1%
全球氣候因溫室效應影響已產生重大的變化，水旱
災、風災以及暴風雪等天災頻傳，並且天災強度與
影響範圍亦逐漸擴增，對企業的永續營運造成潛
在風險。對此，本公司除了承保產物保險及通過
ISO14001環境管理系統外，也積極控管供應鏈的廠
商及客戶，建立相關的緊急應變備援體系，以便在
災害發生時迅速應變，將公司的影響降至最低。本
公司目前制定的排放量減量目標為：每年較上一年
溫室氣體排放量減少1%，溫室氣體減量解決方案如
下：
1.	 全面使用電子化簽核流程，降低紙張使用。
2.	閒置區域照明、空調關閉，僅維持基本動力需求。
3.	管制空調設備，依廠內實際室內溫度彈性調整。
4.	空調、空壓系統加裝變頻節能系統。
5.	排風換氣系統定時管理控制。
6.	更換節能燈具、及推行午休、下班隨手關燈活動。			
7.	辦公室空調溫度設定為26-28度C。

職業安全管理

為防止意外事故之發生及建立緊急

衛生無虞之環境，達成永續發展之目
標。提供良好安全衛生健康舒適的工
作環境，是智邦落實職業安全衛生的
承諾。設置職業安全衛生委員會(安委

工健康檢查，消弭與工作有關的不安
全行為和環境，並持續投入資源，積
極尋求改善與解決各項在工作場所中
潛在的安全衛生問題。因此，為有效
持續推動和管理，訂定執行重點，我
們承諾：
•	
符合國內環安衛法規，有效實施
並持續改善環安衛管理系統。

依循國際OHSAS

加強員工教育及供應商、承包商

自動檢查消弭不安全動作及環

熱心公益的智邦

每天，報章媒體上一件一件，一幕一幕社會救助需求，說明了生命的脆弱與人生的無常。相較於那些因
經濟問題或造化弄人的悲苦故事，能夠安穩平安生活的我們，心存感恩。因此，我們發願，在感謝平安
幸福的同時，也能夠對這塊摯愛的土地，對那些我們認識或不認識的人們，貢獻一份心力。
秉持著「每一顆付出的愛心，將是一粒粒深植的種子，必將長大成蔭，回饋照顧更多需要的人」的理
念，智邦科技在1999年成立了「智邦文教基金會」，我們深信，社會弱勢族群的慈善關懷，是締造人
類文明的希望之窗，而這一扇窗，終將透過社會大眾的愛心參與，讓社會可以重新窺見真、善、美的境
界。我們的公益活動分為3大類，包括(1)照顧社會弱勢(2)關懷弱勢學童(3)員工參與公益善行。
